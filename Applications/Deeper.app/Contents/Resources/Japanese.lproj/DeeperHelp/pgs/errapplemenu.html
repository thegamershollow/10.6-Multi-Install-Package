Inactive items in the Apple menu

If the "Restart", "Shut Down" and "Log Out..." menu items, in the Apple menu don't work, the Kiosk mode is probably turned on:

Open Deeper, and type your password.

Click the Login icon in the toolbar.

Uncheck the "Turn off the Restart, Shut Down and Log Out commands in the Apple menu" case.

Any modification of this function requires the restart of the computer. To restart cleanly, run the Terminal, type "sudo shutdown -r now" and press the Return key, then enter your password and press the Return key again.

This problem sometimes occurs after updating the System.

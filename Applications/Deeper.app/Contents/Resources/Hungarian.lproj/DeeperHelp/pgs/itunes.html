iTunes pane

You can use the iTunes pane to set parameters for the hidden functions of iTunes.

Importing:

■

Add Track Numbers: By default, while importing the songs in iTunes, the name of the created file starts with the track number. You can use this option to display or not display the track number in the name of the created files while importing.

■

Automatically play the songs: You can use this option to automatically start the reading of the imported songs. Generally, the reading only starts after importing the first song.

Misc. Options:

■

Turn off iTunes Store and Ping: You can use this option to turn access to the iTunes Store and social network Ping on or off. The Store menu in the menu bar, the link arrows, and the Ping buttons, as well as the iTunes Store and Ping entries in the left side of the window, will be hidden.

■

Show the Ping buttons: When you select a song, a Ping button appears in the Name column of iTunes lists. You can use this option to show or hide the Ping buttons.

■

Show the link arrows: By default, you can click on a link arrow in iTunes to connect to the iTunes Store. You can use this option to show or hide these link arrows and to choose between the iTunes Store and your iTunes Library.

■

Turn on the rating by half-star: You can assign a rating to a song, album, video, podcast, or other item to indicate how much you like it. You can turn on this option to specify the rating with a half-star. Click just beside a point or a star to assign an intermediate notation (1/2).

Window:

■

Turn on the High Contrast mode: This option replaces the background to the left side of the window as well as all the lists by a black background, which improves the contrast.

■

Horizontally display the control buttons: This function only works with iTunes version 10. By default, the three control buttons of the window (red, yellow, and green) are located vertically. You can use this option to display these buttons horizontally as in all windows.

■

Turn off mini-player with the green zoom button: By default, when you click the green zoom button, the window toggles to the mini-player. You can use this option to maximize the iTunes window. The mini-player mode remains available from the View menu or with the "Shift-Command M" shorcut.

Click on Restore Defaults to cancel all modifications caused by Deeper or by any other application. After execution, the display will be the same as when the System and the applications were first installed.

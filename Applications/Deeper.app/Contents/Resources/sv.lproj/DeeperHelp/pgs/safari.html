Safari pane

You can use the Safari pane to set the parameters of the hidden functions of Safari.

Navigation:

■

Automatically open .PDF-files in Safari: By default, Safari opens and displays the content of PDF files found on the web. Nevertheless, You can turn this behavior off. PDF files will be downloaded.

■

Turn off Safari's image cache: When you browse the Internet, Safari creates two images of the contents of each webpage viewed (a JPEG and another PNG). These images are used as seen when viewing the top sites or s. You can use this option to turn on or turn off the creation of these images.

■

Turn on the Back/Forward buttons: You can use this option to turn on and off the Back and Forward buttons located in Safari's address bar as well as the History menu which appears when the user clicks on these buttons.

■

Turn on single window mode: Some webpages have links which, when clicked upon, will open in a new window. You can use this option to open these pages in a new tab.

■

Confirm when closing multiple pages: You can use this option to turn on or turn off the confirmation message which appears when closing multiple windows or a window containing multiple tabs.

Misc. Options:

■

Turn on Develop menu: Safari has hidden features which you can turn on or turn off with this option. You can use this option to open the current webpage in another browser, change the User Agent, display the Web Inspector, turn on or turn off some functions, and so on. This option also adds an Inspect Element item in the shortcut menu of Safari. By default, the Web Inspector appears in an individual window, but it can be merged with the displayed page by clicking the icon intended for this purpose located at the Inspector's bottom left.

■

Show Bookmark Icons: By default, Safari saves the Bookmark icons of all websites visited. Nevertheless, you can choose not to display these icons by deselecting this option.

■

Show the URL of links in the ToolTips: You can turn on this function to display the complete address of links (webpage and email) in a contextual ToolTip when the mouse hovers above these links.

Printing:

■

Backgrounds: Select this option if you want to print the background of webpages with Safari.

■

Headers and Footers: You can use this option to decide whether you want to print the headers and the footers of webpages.

You can also modify this parameter directly in Safari. (Choose Print in the File menu, then select Safari in the Copies & Pages menu to access the "Print backgrounds" and "Print headers and footers" options.)

Logs:

Enter the values you want, then click the Apply button.

■

Number of items in the History Menu: You can use this field to set the number of items appearing in the History menu in Safari.

■

Number of days: You can use this field to set the number of days preserved in the History of Safari.

Zoom Text Only:

This option in Safari, accessible from the View menu, is disabled as a hidden feature of WebKit. You can use the "Re-enable" button to remove the offending property and fix the bug.

Click on Restore Defaults to cancel all modifications caused by Deeper or by another application. After execution, the parameters will be reset, and the display will be the same as when the System and the applications were first installed.

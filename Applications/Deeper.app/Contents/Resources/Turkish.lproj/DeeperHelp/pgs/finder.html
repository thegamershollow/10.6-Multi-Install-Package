Finder paneli

You can use the Finder pane to adjust parameters relating to the Finder.

Graphic effects:

■

Application zooming: When you double-click on an application icon, a light rectangular zooming effect appears. You can use this option to turn on or turn off this effect.

■

Window zooming: When windows open or close, a zooming effect similar to the Scale effect of the Dock is used. If this effect is turned off, windows will open and close instantly.

■

Animate Info Window: You can use this option to turn on or turn off the effect when you fold/unfold the categories of the Finder's Info window.

■

Animate Snap to grid: You can use this option to turn on or turn off the effect relating to the option "Snap to grid" in the windows in View options as icons.

Misc. Options:

■

Show hidden files and folders: The system hides certain folders (private, usr, bin, sbin etc.) and files (mach_kernel, Desktop, all files whose names begin with a dot etc.). This option makes them visible or invisible in the Finder, but not in volumes and folders placed in the Dock.

■

Show the path in the windows title: This option displays the full path of the selected item in the title bar of the Finder's windows.

■

Show the folder content previews in Quick Look: By default, the folder content with Quick Look displays the icon of selected folder. You can use this option to display a transparent folder icon as well as the preview items located in the selected folder. When the folder contains more items, a small animation rotate the various previews.

■

Label lines: You can use these options to allow an icon's accompanying text to flow in one or two lines. Choosing one of these options will immediately relaunch the Finder.

Menus:

■

Menu Bar Background: You can use this option to modify the background of the menu bar (which is transparent by default). Modifying the background will take effect only after restarting the computer.

You can use Deeper to show or hide various Finder menu entries. As each option is changed, Deeper automatically relaunches the Finder verifying that no other operation is in progress (copying, moving, burning, and emptying the trash).

■

Finder menu: You can use these options to allows you to show or hide the "Preferences...", "Empty Trash", and "Quit Finder" entries in the drop-down Finder menu. The Quit Finder option quits Finder. To reopen the Finder, click the Finder icon in the Dock.

■

File menu: You can use these options to show or hide the Eject and "Burn to Disc" entries in the drop-down File menu.

■

View menu: You can use these options to show or hide the Customize Toolbar and View Options entries in the drop-down View menu.

■

Go menu: You can use these options to show or hide iDisk, "Go to Folder...", and "Connect to Server..." entries in the drop-down Go menu. You can also turn off the "My iDisk" menu entry located in the iDisk submenu of Finder's drop-down Go menu.

Click on Restore Defaults to cancel all modifications caused by Deeper or by any other application. After execution, the display will be the same as when the System and the applications were first installed.

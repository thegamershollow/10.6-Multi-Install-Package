Mail

You can use the Mail tab of the Parameters pane to set the parameters of some hidden Mail functions.

Mail:

■

Display TEXT mails instead of HTML when available: By default, Mail displays the new messages in RTF or HTML format. You can use this option to display them in Text format, without style and without attachment preview. This speeds up the display. To see the attachments, click the Save button located in the header of the message.

■

Turn on Data Detectors: This option turns on and off Data Detectors in Mail. This is a mechanism that allows Mail to recover certain types of information and to propose a suitable action from a local menu.

■

Turn on display attachments in messages: By default, some attachments, including images, are visible directly in the message body. You can use this option to view or not these attachments in messages.

■

Minimum font size: By default, Mail's messages appear with a minimum font size of 10. You can use this option to increase or decrease this size.

■

Hide the name when copying an email address from the shortcut menu: By default, when you copy an email from the Copy Address shortcut, the address is preceded by the name of the recipient. You can use this option to show or hide the name.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by any other application. After execution, the parameters will be reset, and the display will be as it was when the System and the applications were first installed.

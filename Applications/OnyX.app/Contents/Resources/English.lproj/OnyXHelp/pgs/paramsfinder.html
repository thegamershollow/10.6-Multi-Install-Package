Finder

The Finder tab of the Parameters pane allows adjustment of parameters relating to the Finder.

Graphic effects:

■

Application Zooming: When you double-click on an application icon, a light rectangular zooming effect appears. You can use this option to turn this effect on or off.

■

Window zooming: When windows open or close, a zooming effect similar to the Scale effect of the Dock is used. If this effect is turned off, windows will open and close instantly.

■

Animate Info Window: You can use this option to turn on or off the effect when you fold/unfold the categories of the Finder's Info window.

■

Animate Snap to grid: You can use this option to turn on or off the effect relating to the option "Snap to grid" in the windows in View options as icons.

Misc. options:

■

Show hidden files and folders: The system hides certain folders (private, usr, bin, sbin, and so on) and files (mach_kernel, Desktop, all files whose names begin with a dot, and so on). This option makes them visible or invisible in the Finder, but not in volumes and folders placed in the Dock.

■

Show item path in the window title: This option displays the full path of the selected item in the title bar of the Finder's windows.

■

Show the folder content previews in Quick Look: By default, the folder content with Quick Look displays the icon of the selected folder. You can use this option to display a transparent folder icon as well as the preview items located in the selected folder. When the folder contains more items, a small animation rotates the various previews.

■

Label lines: These options allow an icon's accompanying text to flow in one or two lines. Selecting one of these options will immediately relaunch the Finder.

Menus:

■

Menu Bar Background: You can use this option to modify the background of the menu bar (which is transparent by default). Modifying the background will take effect only after restarting the computer.

You can use OnyX to show or hide various Finder menu entries. As each option is changed, OnyX automatically relaunches the Finder while verifying that no other operation is in progress (copying, moving, burning, and emptying the Trash).

■

Finder menu: You can use these options to show or hide the Preferences, Empty Trash, and Quit Finder entries in the drop-down Finder menu. The Quit Finder option quits the Finder. To relaunch, click the Finder icon in the Dock.

■

File menu: You can use these options to show or hide the Eject and "Burn to Disc" entries in the drop-down File menu.

■

View menu: You can use these options to show or hide the Customize Toolbar and View Options entries in the drop-down View menu.

■

Go Menu: You can use these options to show or hide iDisk, "Go to Folder", and "Connect to Server" entries in the drop-down Go menu. You can also turn off the "My iDisk" menu entry located in the iDisk submenu of the Finder's drop-down Go menu.

■

Extras menu: This local menu regroups all the menus (generally available individually from System Preferences); it can be added to the right in the menu bar. Choose the desired menu and click the Show button.
Note: Depending on the configuration, some menus may not appear in the menu bar or be ineffective.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by any other application. After execution, the display will be the same as when the System and the applications were first installed.

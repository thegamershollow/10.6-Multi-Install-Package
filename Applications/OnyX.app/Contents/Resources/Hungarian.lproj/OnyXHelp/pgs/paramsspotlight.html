Spotlight

You can use the Spotlight tab of the Parameters pane to set parameters for the hidden functions of Spotlight.

Options:

■

Indexing the startup disk: You can use OnyX to authorize or prevent the indexing of the startup disk with Spotlight.

■

Search in Dictionary: If you turn off the search in the Dictionary, no Dictionary definitions will appear in the Spotlight search-results.

■

Calculator: By default, you can make simple calculations in the Spotlight search field in the menu bar. You can use OnyX to turn on or turn off this function.

Spotlight's icon:

■

Show/Hide: You can choose to show or hide Spotlight's icon in the menu bar. This does not affect the indexing itself. If you hide Spotlight's icon and repair the permissions with another utility, the icon in the menu bar will reappear.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by another application. After execution, the parameters will be reset, and the display will be the same as when the System and the applications were first installed.

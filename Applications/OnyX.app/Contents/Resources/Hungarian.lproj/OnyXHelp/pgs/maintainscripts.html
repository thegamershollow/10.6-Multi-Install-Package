Execute Maintenance Scripts

Mac OS X includes a task manager (launchd) that automates System cleaning by executing the "periodic" utility. This utility runs three BSD standard scripts which perform many maintenance operations.

To run the Scripts:

Open the Maintenance pane, and click the Scripts tab.

Select the scripts you want to execute (daily, weekly, and monthly).

Click the Execute button.

Note: You can use the Automation pane to run all maintenance scripts at once.

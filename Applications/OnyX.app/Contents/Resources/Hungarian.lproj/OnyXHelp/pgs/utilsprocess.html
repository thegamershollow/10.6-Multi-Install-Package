Process

You can use the Process pane to turn on the Process accounting option, which is turned off by default.

If this option is turned on, Mac OS X will save all processes executed by all the users of the computer in a file (/private/var/account/acct). You can also display and delete the content of this file.

Warning: Many processes are in use, so this file can become very large. You should delete its content occasionally. This operation is launched while deleting the logs (the Automation pane and the Logs command in the Cleaning pane).

Dock

You can use the Dock tab of the Parameters pane to set parameters of some hidden functions of the Dock.

Appearance:

■

Appearance: You can use this menu to modify the appearance of the Dock: 3D with mirror effect (default option) or 2D with transparency effect.

Minimize:

■

Effect: By default, the Dock offers two minimization effects (Genie and Scale). You can use OnyX to turn on a third effect (Suck). Selecting one of these options immediately modifies the parameters and relaunches the Dock. To view the effect in slow motion, hold down the Shift key while minimizing.

■

Reduction proportional: By default, when you minimize a window in the Dock, its preview is proportional. Disabling this function forces the reduction of the preview icon unproportionally (128 pixels x 128 pixels).

Position and alignment:

■

Position: You can use this option to place the Dock at the bottom, left, or right side of the screen. Selecting one of these options immediately modifies the parameters and relaunches the Dock.

■

Alignment: You use this option to center or anchor the Dock to the edges of the screen. Selecting one of these options immediately modifies the parameters and relaunches the Dock.

Graphic effects:

■

Opening applications: Enabling this option will make the icon bounce in the Dock when you open an application.

■

When an alert message appears in the background: When an application is running in the background and solicits your intervention (appearance of alert message, request of password, and so on), its icon bounces in the Dock at regular intervals. You can use this option to remove this behavior.

Misc. options:

■

Only show open applications: Check this option to display only the icons of open applications in the Dock.

■

Use transparent icons for hidden applications: This option allows the icons of hidden applications to appear as transparent.

■

Automatically hide and show the Dock: If you turn on this option, the Dock will be hidden when you don't use it. To display the Dock, move the pointer to the bottom or an edge of the screen.

■

Turn off animation: Use this option to turn off the graphic effect when showing/hiding the Dock.

■

Turn on the Exposé menus: By default, a long click (or left click) on the icon of an application located in the Dock, displays windows of the application as small thumbnails (as the F10 or fn+F10 Exposé function) and also displays a horizontal button bar with the Hide, Quit, and Options commands. Turning off this option will display a classic menu (Hide, Quit, and Options) and the list of application windows.

■

Lock icon size: This option prevents resizing icons in the Dock.

■

Lock Dock Content: This option prevents moving and removing icons in the Dock.

■

Turn on the Single Application mode: In the Dock, when you click the icon of an application already launched, it appears in the foreground. Turning on this mode will hide all other applications simultaneously.

Stacks:

■

Non hierarchical List menu: This option displays a different kind of List menu, non hierarchical, in the stacks located in the right side of the Dock. You can zoom in or zoom out the menu's size using the shortcuts Command + and Command -. There are five different sizes available.

■

Turn on mouse-over on stacks displayed in grid-mode: When a stack displays its content as grid, the items are framed when the mouse passes above.

■

Add Recent/Favorite items stack: Click the "Add" button to create a Recent/Favorite items stack. This stack has a special function. In the Dock, Control-click its icon or right-click, and select the items you wish to appear. Then, click again on its icon to display the previously selected items as grid (Recent Applications, Recent Documents, Recent Servers, Favorite Volumes, or Favorite Items).

Add spacing between the icons:

Invisible spaces can be added in the Dock.
These spaces are moveable and can be removed like any icon.

■

Applications: One or more invisible spaces can be added to the left side of the Dock by clicking on the "Applications" button.

■

Others: One or more invisible spaces can be added in the right side of the Dock by clicking on the "Others" button.

Click the Restore Defaults button to reset the parameters and to cancel all modifications caused by OnyX or by another application.

Click on Restore Dock to reset the parameters and to restore the display. The display will be the same as when the System and the applications were first installed.

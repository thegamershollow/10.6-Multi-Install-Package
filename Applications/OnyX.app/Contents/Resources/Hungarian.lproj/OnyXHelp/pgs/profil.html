Profile

You can use the Profile tab of the Info pane to show or save the profile of your hardware and software configuration.

Three styles of profile exist:

- Mini Profile: This profile contains only minimal information and omits the identification information of your computer (serial number, IP address, and so on).

- Basic Profile: This profile contains most of the hardware information while omitting most of the software information.

- Full Profile: This profile contains all the information of your hardware and software configuration.

Every profile can be saved to the Text format (.txt file readable by TextEdit or any other text editor) or to the XML format (.spx file readable by the System Profiler application).

To show a profile:

Choose the profile you want to show in the "Style of Profile" local menu.

Click the Show button.

To save a profile:

Choose the profile you want to save in the "Style of Profile" local menu.

Choose the format (Text or XML) in the Format local menu.

Click the Save button. The Profile file is saved on your Desktop.

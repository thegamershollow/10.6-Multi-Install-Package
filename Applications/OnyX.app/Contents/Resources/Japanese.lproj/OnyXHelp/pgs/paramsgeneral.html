一般

The General tab allows adjustment of some global parameters.

Windows:

■

スクロールアロー： You can use OnyX to choose among four types of arrows in window scroll bars. Select one of these options to modify the parameters immediately and relaunch the Finder; the applications already in memory must be relaunched to show the changes.

■

シートダイアログの表示速度： You can use this option to set the speed that a window's sheet is displayed. A "sheet" is a modal dialog which is attached to a window and asks for input from the user.

CrashReporter:

■

CrashReporter の動作モード： By default, when an application quits unexpectedly, an alert message appears that allows you to reopen the application and send a report to Apple. It's the basic mode of operation of Mac OS X, but two other modes exist. The Developer mode displays an alert message with miscellaneous debug information. The Server mode, turned on by default in Mac OS X Server, doesn’t display any message but stores the Crash reports in the Library/Logs/CrashReporter and Library/Logs/CrashReporter folder in your home folder.

ネットワークボリューム：

■

.DS_Store ファイルを作成しない： You can use this option to prevent .DS_Store files creation on network volumes. By default, these files are invisible under Mac OS X, but they become visible (and they are useless) on other operating systems.

Screen Captures:

■

スクリーンキャプチャの形式： You can use this menu to change the format (PNG by default) of Screen Captures made by using the keyboard shortcuts. You may choose among nine formats: BMP, GIF, JPEG, JPEG 2000, OpenEXR, PDF, PNG, TGA, and TIFF.

■

ウインドウの影もキャプチャ： By default, when you capture a window (Shift + Command + 4 then Space bar), the shadow of this one is also captured. You can use this option to exclude the shadow of window captures.

■

パス： This option specifies the saving-path for screenshots (your Desktop, by default). To change this, click the Select button, choose the folder, and click the Choose button.

■

名前： You can choose the name of the screen captures by typing the name in the Name field and clicking the Apply button.

Recent items:

■

アプリケーションのファイルメニューの“最近使った項目”に項目を表示： The majority of the applications that allow you to open files have an Open Recent menu item in their File menu. By default, this submenu displays a list of the ten last documents opened by the application. You can use this function to turn the appearance of this list on or off. However, certain applications (Photoshop, GoLive, GraphicConverter, and so on) use their own routine to store Recent Items.

■

最近使った項目の数： You can use this field to set the number of entries in the Open Recent command of the File menu of applications that allow it.

■

開く／保存ダイアログの最近使った場所の数： This local menu allows you to display between 1 and 12 recent places (the default is 5) in the Open/Save dialogs.

Animated Desktop Background:

■

スクリーンセーバをデスクトップに表示： You can use this option to choose a Screen Saver among those installed by Mac OS X and start it as a Desktop Background.

Downloaded applications:

■

ダウンロードしたアプリケーションの初回起動時に表示される警告を削除： At first launch of an application downloaded from the Internet, an alert message appears. This message gives miscellaneous information on the source of the downloaded file, and you can open the file. You can use this option to show or hide this alert message.

Preferences Panes:

■

アーカイブ： This button launches the installation of an Archives Preferences pane that allows changing the settings of the Archive Utility application.

■

プロセッサ： This button launches the installation of a Processor Preferences pane.

To uninstall a Preferences pane, click its icon with the right button of the mouse or hold the Control key while clicking the icon with the left mouse button.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by another application. After execution, the parameters will be reset, and the display will be the same as when the System and the applications were first installed.

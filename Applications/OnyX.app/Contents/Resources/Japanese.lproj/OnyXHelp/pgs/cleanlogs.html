Logs

When an application is open, many messages are run between the application, the system, and its components. This information is stored in log files. The system also creates logs and archives them during system maintenance operations. When an application does not respond, the details are saved in a diagnostic report that you can use to diagnose the problem, or you can send to the developer (if asked to do so).

You can use the Automation pane to remove all logs and diagnostic reports except the OnyX, bash, and System Software Update Logs.

You can use the Logs tab of the Cleaning pane to individually delete the logs and diagnostic reports.

■

Log files: This option deletes the applications Logs, Terminal Command Log, and some System Logs.

■

Apple Software Update log: You can delete the preference list files that contains the Software Update log using this option. This file (InstallHistory.plist) is located in the /Library/Receipts/ folder and can viewed in the Software Update preferences pane (Installed Updates tab).

■

bash Log: You can delete the log that contains the list of commands entered in Terminal using this option.

■

System Archived logs: When the periodic scripts (daily, weekly, and monthly) are run, log files are archived and compressed. There may be up to seven archives per log. You can remove all of the compressed archives and some System-saved logs using this option.

■

Instant message logs: This option removes the saved conversations of the most popular Instant Messaging applications. Supported applications: Adium, AIM, aMSN, Colloquy, Conversation, Fire, iChat, iRC, JBuddyMessenger, jIRCii, MacOSX.com Chat, MChat, Microsoft Messenger, Minerva, Proteus, Psi, Skype, Snak, SnapTalk, Trillian, VChat, X-Chat Azure, and Yahoo! Messenger.

■

User Diagnostic Reports: You can use this option to delete all User Diagnostic Reports. The daily maintenance script also deletes these files, but only those that have not been modified in the last 60 days. Any web browser creates a FullCircle folder with two files: manifest.ini and permdata.box. These files can be sent when a browser quits unexpectedly and will be used by developers to diagnose and correct the cause of the problem. Each new version adds a new folder--making the previous one outdated. This option deletes the FullCircle folder located in the /Library/Application Support folder in your home folder.

■

System Diagnostic Reports: This option deletes any Diagnostic Reports created by the system, and relaunches the syslogd daemon (process that records the activities of the system).

■

Mobile Devices CrashReporter: This option deletes any Diagnostic Reports that have occurred on your mobile device.

Fonts Cache

The Fonts tab of the Cleaning pane and the Automation pane allows you to remove the Fonts Cache files.

You can use these files to accelerate font loading. Deleting them will temporarily slow the fonts from appearing, but it will also correct some problems. This will not delete any personal font collection.

■

From the Fonts tab of the Cleaning pane, you can independently remove the Fonts Cache for the System and Users, Adobe, Microsoft, Quark, and of some Open Source applications.

■

From the Automation pane, you can delete all Fonts Caches.

Note: After cleaning the Fonts Cache of some applications, the initial startup can be particularly lengthy.

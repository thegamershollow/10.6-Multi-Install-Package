Manuals

Mac OS X makes use of several hundred Unix commands. These commands are accessible in Terminal and are very well documented in the Manuals. You can use the man command in the Unix Utilities pane to format and display these manual pages.


To display a "man" page:

Click the Utilities icon in the toolbar, and select the Manuals tab.

Select the name of the command in the list at left, or type the name of the command in the search field.

Click the Show button.

To convert the man page to PDF and open it automatically with Preview, click the Create PDF button.

You can use the Visibility pane to hide or display the whatis.db database, located in the /usr/share/man/ folder used by the "whatis", "apropos", and "man" commands.

Applications

You can use the Applications pane to execute applications present by default in Mac OS X but not easily accessible to you.

画面共有:

Mac OS X v10.5、ネットワーク上の別のコンピュータに接続し、そのコンピュータの画面を自分のコンピュータに表示できます。コンピュータの画面を共有している間は、そのコンピュータ上で発生することを制御できます。たとえば、書類やアプリケーションを開いたり、ウインドウを開いたり、移動したり、閉じたり、コンピュータをシステム終了したりすることができます。

ネットワークの別のコンピュータで自分のコンピュータの画面を共有することもできます。

ネットワーク診断:

「ネットワーク診断」を使用して、インターネット接続の問題の解決に役立てることができます。たとえば、インターネットに接続できない場合は、「ネットワーク診断」を開いて、問題を解決することができます。

「ネットワーク診断」を使用して、「Ethernet」、「AirMac」、内蔵または外部モデムの接続に関する問題を診断することができます。別の方法でインターネットに接続しようとしている場合は、インターネットアカウントに付属しているマニュアルを参照してください。

文字ビューア:

数学記号、アクセント記号付き文字、矢印やその他の飾り文字などは、「文字ビューア」を使用して書類に入力します。

また、「文字ビューア」を使用すると、日本語、繁体字中国語、簡体字中国語、および韓国語の各文字のほか、その他の言語の文字も入力できます。

The Character palette is available in most applications from the Special Characters command in the Edit menu.

ディレクトリユーティリティ:

「ディレクトリユーティリティ」は、Mac OS X コンピュータが使うディレクトリサービス、および特定のディレクトリドメインに接続する方法を設定します。 また、特定のディレクトリドメインの認証および連絡先情報を検索するための検索方式も定義します。

See the Directory Utility Help for more info.

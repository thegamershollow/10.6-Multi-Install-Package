Forgot Password

管理者パスワードを忘れた場合は、Mac OS X のインストールディスクを使用してパスワードをリセットできます。

Mac OS X のインストールディスクをセットし、コンピュータを再起動します。

起動音が聞こえたら、回転する歯車が表示されるまで C キーを押したままにします。

「Language Chooser」が表示されたら、言語を選択して「続ける」をクリックします。

インストーラで、「ユーティリティ」＞「リセットパスワード」と選択します。

画面の指示に従ってパスワードを変更します。

ログイン時にキーチェーンがロック解除されるように、新しいアカウントパスワードと一致するようにログインキーチェーンのパスワードを変更してください。

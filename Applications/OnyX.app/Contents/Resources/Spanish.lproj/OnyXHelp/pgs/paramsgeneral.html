General

La pestaña General le permite ajustar parámetros globales.

Ventanas:

■

Flechas en la Barra de Scroll: OnyX le permite elegir entre cuatro tipos de flechas en la barra de scroll de las ventanas. Seleccione una de estas opciones para modificar los parámetros inmediatamente y reiniciar el Finder; las aplicaciones que esten en memoria deben ser reiniciadas para mostrar los cambios.

■

Velocidad de visualización de ventanas: Esta opción le permite definir la velocidad con la que las hojas de las ventanas se mostrarán. Una hoja es un diálogo modal adjunto a una ventana que pregunta algo al usuario.

CrashReporter:

■

Modo de operación del CrashReporter: Por defecto, cuando una aplicación falla, aparece un mensaje de aviso dándole la opción de reiniciar la aplicación y enviar un informe a Apple. Este es el modo básico de operación del Mac OS X, pero existen otros dos modos. El modo Desarrollador muestra un mensaje de aviso con diversa información de depuración. El modo Servidor, activado por defecto en Mac OS X Server, no muestra ningún mensaje pero almacena los informes de fallo en las carpetas Library/Logs/CrashReporter y Library/Logs/CrashReporter en su carpeta Inicio.

Volúmenes de red:

■

Desactivar la creación de ficheros .DS_Store: Esta opción le permite impedir la creación de ficheros .DS_Store en volúmenes de red. Por defecto, estos ficheros son invisibles en Mac OS X pero no para otros sistemas operativos.

Capturas de Pantalla:

■

Formato de captura de pantalla: Este menú le permite modificar el formato (PNG por defecto) de las capturas de Pantalla usando los atajos de teclado. Elija el formato deseado de entre los nueve formatos disponibles: BMP, GIF, JPEG, JPEG 2000, OpenEXR, PDF, PNG, TGA y TIFF.

■

Sombra en las capturas de ventana: Por defecto, cuando captura una ventana (Shift + Comando + 4 y Barra espaciadora), la sombra de esta también es capturada. Esta opción le permite excluir la sombra de las capturas de ventana.

■

Ruta: Este campo indica la ruta completa de las capturas de pantalla (El Escritorio del usuario, por defecto). Para modificar esta ruta, pulse el botón Seleccionar, elija la carpeta deseada y pulse el botón Elegir.

■

Nombre: Usted puede elegir el nombre de las capturas de pantalla introduciendo el nombre deseado en el campo Nombre y pulsando el botón Aplicar.

Ítems recientes:

■

Activar visualización de ítems en la opción de menú Archivo > Abrir Recientes de aplicaciones: La mayoría de las aplicaciones que le permiten abrir documentos tienen una opción de menú "Abrir Recientes" en su menú Archivo. Por defecto, este submenú muestra la lista de los diez últimos documentos abiertos por la aplicación. Esta función le permite activar o desactivar la visualización de esta lista. Sin embargo, esto no es aplicable a ciertas aplicaciones (Photoshop, GoLive, GraphicConverter...) porque estas usan sus propias rutinas para almacenar los ítems recientes.

■

Número de ítems recientes: Este campo le permite definir el número máximo de entradas en la opción de menú "Abrir Recientes" de las aplicaciones.

■

Número de Sitios Recientes en los diálogos Abrir/Guardar: Este menú local le permite mostrar entre 1 y 12 sitios recientes (por defecto son 5) en los diálogos Abrir/Guardar.

Fondo de Escritorio Animado:

■

Fondo de Escritorio Animado: Esta opción le permite elegir un Salvapantallas de entre los instalados por Mac OS X e iniciarlo como Fondo de Escritorio.

Aplicaciones descargadas:

■

Suprime el mensaje de aviso de la primera ejecución de aplicaciones descargadas: Al ejecutar por primera vez una aplicación descargada de Internet, aparece un mensaje de aviso. Este mensaje ofrece diversa información del origen del fichero descargado y le permite cancelar la ejecución. Esta opción le permite mostrar u ocultar este mensaje de aviso.

Paneles de Preferencias:

■

Archivos: Este botón ejecuta la instalación del panel de Preferencias Archivos que le permite modificar las configuraciones de la aplicación "Utilidad de Archivos".

■

Procesador: Este botón ejecuta la instalación del panel de Preferencias Procesador.

Para desinstalar los paneles de Preferencias, pulse su icono con el botón derecho del ratón o manteniendo pulsada la tecla Control...

Pulse en Restaurar valores por omisión para cancelar todas las modificaciones hechas con OnyX o por cualquier otra aplicación. Después de ejecutarlo, los parámetros serán reiniciados y mostrados tal cual se definieron en la primera instalación del Sistema y las aplicaciones.

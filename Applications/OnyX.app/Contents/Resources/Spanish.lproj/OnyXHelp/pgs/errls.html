Problema con la opción "Abrir con" del menú contextual del Finder

Si la opción "Abrir con" del menú contextual del Finder muestra una lista de aplicaciones con duplicados o con aplicaciones desinstaladas, OnyX puede ayudarle corrigiendo el problema. Es aconsejable reconstruir la base de datos de LaunchServices.

Ejecute OnyX.

Abra el panel Automatización o abra el panel Mantenimiento y seleccione la pestaña Reconstruir...

Compruebe que está seleccionado "LaunchServices" y no seleccione las otras opciones.

Pulse el botón Ejecutar.

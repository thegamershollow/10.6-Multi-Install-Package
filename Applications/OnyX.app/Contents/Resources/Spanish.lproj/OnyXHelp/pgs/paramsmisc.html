Otros

La pestaña Otros del panel de Parámetros le permite configurar parámetros de funciones ocultas de aplicaciones.

Utilidad de Discos:

■

Verificar Imágenes de disco (.dmg): Esta opción le permite activar o desactivar la verificación cuando usted monta Imágenes de Disco.

■

Añadir otros formatos de Imágenes de Disco: Esta opción le permite añadir formatos suplementarios cuando desee convertir una imagen de disco ocrear una imagen de disco desde una carpeta. Esta función no es operativa a la hora de crear una nueva imagen de disco vacía.

■

Activar el menú Debug: Seleccione esta opción si desea mostrar el menú Debug en la barra de menús de la Utilidad de Discos.

■

Verificar los discos después de grabar: Esta opción le permite activar o desactivar la verificación después de grabar un disco.

Visor de Ayuda:

■

Configurar ventanas en modo flotante: Por defecto, la aplicación del Visor de Ayuda muestra una ventana flotante, esto significa que siempre está en primer plano, sobre cualquier otra ventana que estuviese abierta. La desactivación de esta opción le permite mostrar esta ventana en segundo plano y que sea ocultada parcial o totalmente por otras ventanas. When this option is turned off, the Help Viewer icon is visible in the Dock and in the Application Switcher.

■

Mostrar el icono en el Dock y en el conmutador de aplicaciones: Por defecto, cuando se consulta la Ayuda de Mac y de muchas de las aplicaciones, el icono del Visor de Ayuda no aparece ni en el Dock ni en el conmutador de aplicaciones (Comando + Tab). Esta opción le permite mostrar u ocultarlo.

■

Activar el modo Debug: Este modo añade diversa información en el registro de Sistema (/private/var/log/system.log) mientras usa la aplicación del Visor de Ayuda.

X11:

■

Mostrar un aviso antes de salir de la aplicación: Esta opción le permite mostrar u ocultar el mensaje de aviso que aparece por defecto al salir de la aplicación X11.

■

Al lanzar X11, ejecutar: Este menú le permite seleccionar la aplicación que desea ejecutar automáticamente al iniciar X11.

Compartir Pantalla:

■

Mostrar el navegador Bonjour: Esta opción le permite mostrar u ocultar la lista de equipos disponible mediante el protocolo Bonjour de Apple.

■

Mostrar la barra de herramientas completa: Esta opción le permite mostrar u ocultar la barra de herramientas completa mientras comparte la pantalla.

Dashboard:

■

Activar Dashboard y sus Widgets: Si usted activa esta opción, al pulsar sobre el icono del Dashboard en el Dock se mostrarán los widgets.

■

Activar el Modo Desarrollador del Dashboard: Esta opción le permite situar y usar los widgets del Dashboard en el Escritorio de su Mac. Seleccione esta opción y pulse el botón Aplicar (el Dock será reiniciado). A continuación, en el menú Manzana, seleccione "Preferencias del Sistema...", pulse sobre el icono Dashboard y Exposé y configure los Atajos de Teclado del Dashboard.

- Para poner un widget en el Escritorio, pulse sobre el icono Dashboard en el Dock o pulse el Atajo de Teclado previamente configurado para mostrar el Dashboard. Pulse sobre el widget deseado y con el botón del ratón pulsado (como si quisiera eliminarlo) pulse el Atajo de Teclado para ocultar el Dashboard. El widget permanecerá en el Escritorio pudiéndo ser usado como cualquier otra aplicación. De este modo, usted puede poner diversos widgets en el Escritorio de su Mac.

- Para eliminar un widget del Escritorio, pulse sobre el widget deseado y con el botón del ratón pulsado pulse el Atajo de Teclado para mostrar el Dashboard. A continuación suelte el botón del ratón y pulse el Atajo de Teclado para ocultar el Dashboard.

■

Turn off the ripple effect when adding a new widget: You can use this option to turn on or off the ripple effect that appears when you add a new widget in Dashboard.

Otras aplicaciones:

■

Apple Remote Desktop: Esta opción le permite mostrar u ocultar el menú Debug en la barra de menús de Apple Remote Desktop.

■

iCal: Esta opción muestra el menú Debug en la barra de menús de iCal.

■

iChat: Esta opción activa y desactiva los Detectores de Datos en iChat. Este es un mecanismo que permite a iChat recuperar algunos tipos de información y proponer una acción adaptada desde un menú local.

■

Terminal: Activa el Foco de ventana en el Terminal activándose la selección de ventana de Terminal cuando se pasa el puntero del ratón sobre ella.

■

Interface Builder: Esta opción, desactivada por defecto desde la versión 3.2, le permite activar el Inspector AppleScript en la aplicación Interface Builder.

Pulse en Restaurar valores por omisión para cancelar todas las modificaciones hechas con OnyX o por cualquier otra aplicación. Después de ejecutarlo, los parámetros serán reiniciados y mostrados tal cual se definieron en la primera instalación del Sistema y las aplicaciones.

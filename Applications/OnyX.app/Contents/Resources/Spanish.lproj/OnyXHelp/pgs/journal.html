Journaling

You can use this option to turn the journaling of the disk on or off.

Si el journaling está activado para un disco, Mac OS X mantiene un registro contínuo de los cambios hechos a los ficheros del disco. Si su ordenador se para debido a un fallo de corriente o cualquier otra causa, Mac OS X utiliza el registro (journal) para recuperar el último estado aceptable del disco duro antes de la parada.

Note: If journaling is turned off, it's not possible to verify the structure of system files on the startup volume.

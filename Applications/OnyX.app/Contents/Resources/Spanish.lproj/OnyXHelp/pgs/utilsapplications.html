Applications

You can use the Applications pane to execute applications present by default in Mac OS X but not easily accessible to you.

Compartir pantalla:

Con el Mac OS X, puede conectarse a otro ordenador de su red y visualizar la pantalla de dicho ordenador en el suyo propio. Mientras comparte la pantalla del ordenador, puede controlar lo que sucede en dicho ordenador como, por ejemplo, abrir documentos o aplicaciones, abrir, mover o cerrar ventanas, o incluso apagar el ordenador.

También puede compartir la pantalla de su ordenador con otros usuarios de la red.

Diagnóstico de Red:

Diagnóstico de Red permite solucionar problemas de conexión a Internet. En caso de que resulte imposible conectarse a Internet, Diagnóstico de Red se abrirá y le indicará los pasos que debe seguir para solucionar el problema.

Diagnóstico de Red permite diagnosticar problemas relacionados con las conexiones Ethernet, AirPort o módem interno o externo. En caso de que se produzcan problemas al intentar conectarse a Internet mediante otro método, consulte la documentación que se incluye con la cuenta de Internet.

Visor de Caracteres:

Utilice el Visor de Caracteres para introducir caracteres especiales y símbolos, como símbolos matemáticos, letras con acento, flechas y otros símbolos especiales ("dingbats"), en los documentos.

El Visor de Caracteres también se puede utilizar para introducir caracteres de los idiomas japonés, chino tradicional, chino simplificado y coreano, además de caracteres de otros idiomas.

The Character palette is available in most applications from the Special Characters command in the Edit menu.

Utilidad de Directorios:

La aplicación Utilidad de Directorios determina qué servicios de directorio utiliza un ordenador Mac OS X y cómo se conecta a dominios de directorio específicos. También define las políticas de búsqueda para buscar información de autenticación y contactos en dominios de directorio específicos.

See the Directory Utility Help for more info.

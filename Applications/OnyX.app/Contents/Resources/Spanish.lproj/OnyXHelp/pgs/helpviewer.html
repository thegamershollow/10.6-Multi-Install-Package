Visor de Ayuda

La aplicación de Visor de Ayuda dispone de un botón Inicio que muestra el listado de las ayudas consultadas. Esta opción le permite reiniciar este menú.

Para encontrar de nuevo la Ayuda de una aplicación en la Biblioteca de Ayuda de Mac OS, usted debe volver a ejecutar la Ayuda, al menos una vez, después de ejecutar esta aplicación.

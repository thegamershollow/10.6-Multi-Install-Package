重要信息

这里有一些重要信息是您在使用 OnyX 之前绝对要了解的。

软件许可协议：

在使用此软件之前请仔细阅读这份软件许可协议。

OnyX 是一个免费，自由使用的软件 (免费软件)。您可以按自己的意愿拷贝及分发此软件。您想使用多久都可以。

您清楚明白并同意 OnyX 软件是按您自己的意愿使用并承担整个使用过程的体验质量、性能、精确性以及后果和风险。

OnyX 软件按照 "意愿使用 (AS IS)" 的原则发布，不做任何类型的责任担保。

作者本人不对事件产生的个人伤害、或偶然、特殊、间接或随之发生的，包括无限制、个人文件及数据丢失、业务损害或任何经济损失及关于使用 OnyX 软件不当造成的损害负责。

备份您的数据：

在使用 OnyX 之前，备份您的数据。

OnyX 不是毫无错误的，而且没有哪个用户的操作永远可靠。使用 Time Machine 或第三方备份工具备份您的数据。

验证启动磁盘：

在运行维护或清理任务之前，强烈推荐您检查启动磁盘的 S.M.A.R.T. 状态及其系统文件。

这些验证都很必要，可以在 OnyX 启动时或从 "验证" 面板进行。系统文件结构的错误比较常见，如果不能修复则很容易造成数据损坏。

电脑无响应：

OnyX 没能力摧毁系统。

如果您的电脑在验证宗卷时失去响应，这是完全正常的行为！一旦 OnyX 完成了检查，控制权就会还给您。

个人数据移除：

OnyX 不会移除您的个人数据。

...除非您完全不明白文件系统的基本规则。Mac OS X 是基于 Unix 的操作系统，它有着完善定义的结构。您的数据必须存储在您的个人文件夹下现成的文件夹内。例如，'文稿'、'图片'、'音乐' 等等...，或之前在您个人文件夹内创建的文件夹内。绝不要存储您的文稿或程序到操作系统专用的文件夹内，例如 '资源库'、'系统'，或这些文件夹内的子文件夹。

任何人都会犯错误，或不正确的处理数据...

请定期备份您的数据。

隐藏功能的问题：

OnyX 不会经常修正会导致意外行为的隐藏功能。

事实上，在参数面板的功能都是由系统或程序开发者或创建的。OnyX 只不过是使用一条简单的命令，按照您的意愿启用和禁用这些功能。如果这些功能导致了一些问题，恢复默认设置。不要忘记如果一个功能默认被禁用，那必然是有原因的...

更新：

经常更新 OnyX 和您的系统。

最新的版本一般都会修正前一版本遇到的部分问题。

卸载 OnyX：

要卸载 OnyX，转到 OnyX 菜单并选择 "卸载 OnyX..."。显示确认消息时，点按继续按钮。应用程序和安装使用时创建的组件都会被移除。。

要取消启动 OnyX 时此警告的显示，可以前往偏好设置 >"消息和日志"，取消选择 "显示重要信息" 选项。

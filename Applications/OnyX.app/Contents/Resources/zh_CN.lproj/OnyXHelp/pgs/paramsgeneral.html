通用

通用标签页允许调整全局参数。

窗口：

■

滚动条的箭头：OnyX 允许您选择窗口滚动条的四种类型。选择一个就会立即修改参数并重启 Finder；已开启的应用程序必须重启才能应用更改。

■

下拉窗显示速度：该选项允许您设置下拉窗的显示速度。下拉窗是与某特定的窗口相关联的对话框，它从窗口标题下滑出并浮在窗口上。

崩溃报告：

■

系统崩溃报告操作模式：默认情况下，当应用程序崩溃，会出现一个提示询问您重新打开程序或发送报告给 Apple。这是 Mac OS X 的基本操作模式，还有两个其它操作模式。开发者模式将显示有其它调试信息的提示。服务器模式，默认在 Mac OS X Server 中启用，不显示通知但储存崩溃报告到 Library/Logs/CrashReporter 和您的个人文件夹下的 Library/Logs/CrashReporter 文件夹。

网络宗卷：

■

禁用 .DS_Store 文件的创建： 这个选项可以防止在网络宗卷上创建 .DS_Store 文件。默认情况下这些文件在 Mac OS X 中不可见但它们在其它操作系统中可见 (而且没用)。

屏幕截图：

■

屏幕截图格式：该菜单允许您更改使用快捷键进行屏幕截图的格式 (默认是 PNG)。可以选择九种格式 (PNG，TIFF，JPEG，JPEG2000，PDF，GIF，OpenEXR，BMP 和 TGA)。

■

窗口截图的阴影：默认情况下您对窗口截图 (Shift + Command + 4 然后空格键)，窗口的阴影也会截下来。这个选项可以排除对阴影的截图。

■

路径：这里显示了屏幕截图存储的完整路径 (默认是用户的桌面)。要更改路径，点按选择按钮，选择指定的文件夹。您也可以在文本框内直接输入路径并点按应用按钮。

■

名称：您可以输入想要的屏幕截图名称并点按 "应用" 按钮。

最近使用项目：

■

启用在应用程序的 "文件 > 打开最近使用的" 项目： 大多数的应用程序的文件菜单都有 "打开最近使用的" 项目。默认情况下，该子菜单显示了十个由程序打开的文档列表。该功能允许您启用或停用该列表的显示。同时，这不适用于特定的程序 (Photoshop，GoLive，GraphicConverter...) 因为它们使用的是自己特有的方式来储存最近使用项目。

■

最近项目个数：此处允许您设置应用程序的文件 > 打开最近的项目中项目最大数量。

■

打开/存储对话框中最近位置数量：这个本地菜单允许您在打开/存储对话框中显示 1 到 12 个最近的位置 (默认为 5)。

动画桌面背景：

■

动画桌面背景：该选项允许您使用 Mac OS X 内建的屏保作为桌面背景。

下载的程序：

■

移去下载的程序首次启动的警告消息： 从网上下载的程序首次启动时会出现一个警告消息. 这个消息给出了下载源的一些信息并可以取消启动. 这个选项可以显示或隐藏该消息.

偏好设置面板：

■

归档：此按钮将安装归档的偏好设置面板以便于修改 "归档实用工具" 的设置。

■

处理器：此按钮将安装处理器偏好设置面板。

要卸载偏好设置面板, 右键点按其图标或按住 Control 键点按...

点按 "恢复默认设置" 可以取消 OnyX 或其它程序所做的更改。执行后，参数将被重置且系统和程序的显示会还原为首次安装的状态。

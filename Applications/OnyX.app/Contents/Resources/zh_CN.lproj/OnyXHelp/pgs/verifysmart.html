检查磁盘的 S.M.A.R.T. 状态

OnyX 在启动时允许您检查并验证启动磁盘的 S.M.A.R.T. 状态。该项操作可预防失败。大多数磁盘都包含 S.M.A.R.T. 硬件功能 (自我监视分析和报告技术) 用以测试和监视磁盘。

警告：检查启动磁盘的 S.M.A.R.T. 只适用于 ATA 或串行 ATA 磁盘。FireWire, USB 和 SCSI 磁盘不被 Mac OS X 支持。OnyX 不能检查 RAID 设备的 S.M.A.R.T. 状态。
- If FileVault is turned on, checking the S.M.A.R.T. status will not be possible (Message: The S.M.A.R.T. status is not supported.). If you turn off FileVault, restart the computer again before checking the S.M.A.R.T. status of the disk.

运行 OnyX 并输入您的管理员密码。

点按 "验证" 图标并选取 S.M.A.R.T. 状态标签页。

点按 "检查" 按钮立即验证启动磁盘的 S.M.A.R.T. 状态。

验证面板的 S.M.A.R.T. 状态项目显示了 S.M.A.R.T. 状态及上次验证的日期。

默认情况下，OnyX 在启动时会询问您是否检查启动磁盘的 S.M.A.R.T. 状态。您可以在 OnyX 偏好设置面板的高级选项中选择自动检查或不进行检查。

其它

参数面板的其它标签页允许您设置应用程序的隐藏功能参数。

磁盘工具：

■

验证磁盘镜像 (.dmg)：该选项允许您启用或停用装上磁盘镜像时的验证过程。

■

添加其它磁盘镜像格式：该选项允许您在转换磁盘镜像或从文件夹创建镜像时添加其它格式。该选项在创建空镜像时无效。

■

显示调试菜单：如果您想显示磁盘工具的调试菜单可以勾选此处。

■

刻录完毕后验证光盘：该选项允许您启用或停用刻录光盘后的验证过程。

帮助查看器：

■

设置窗口为置顶模式: 默认情况下帮助查看器程序是 "置顶窗口", 位于其它程序的前面. 禁用此选项可以让此窗口与其它窗口一样被遮挡。当此项禁用后，帮助查看器图标将在 Dock 中和应用程序切换器可见。

■

在 Dock 和程序切换器中显示图标：默认情况下当查看 Mac 帮助和其它程序帮助时，帮助查看器不在 Dock 也不在程序切换器中显示 (Command + Tab)。这个选项可以显示或隐藏它。

■

启用调试模式：这个模式在使用帮助查看器时添加了系统日志的很多信息 (/private/var/log/system.log)。

X11：

■

退出程序前显示警告消息：此选项允许在退出 X11 程序时显示或隐藏警告信息。

■

启动 X11 时执行：这个菜单可以选择在启动 X11 时自动执行的程序。

屏幕共享：

■

显示 Bonjour 浏览器：这个选项可以显示或隐藏 Apple Bonjour 协议下可用的机器列表。

■

显示完整工具栏：这个选项允许显示或隐藏屏幕共享时的完整工具栏。

Dashboard：

■

启用 Dashboard 和 widget：如果启用该选项，点按 Dock 内的 Dashboard 图标将显示 widget。

■

启用 Dashboard 的开发者模式：该选项允许你在桌面上放置和使用 Dashboard 的 widget。选择该选项并点按应用按钮 (Dock 将会重新装入)。然后，在 Apple菜单，选择 "系统偏好设置..."，点按 Dashboard 和 Exposé 图标并设置 Dashboard 的键盘快捷键。

- 要将 widget 放置到桌面，点按 Dock 内的 Dashboard 图标或按下显示 Dashboard 的键盘快捷键。点按想要的 widget 并按住鼠标键然后按下键盘快捷键隐藏 Dashboard。widget 就仍然留在桌面并可像程序一样使用。用这个方法，你可以放置很多 widget 到桌面上。

- 要从桌面移去 widget，按住所选 widget 并按下快捷键显示 Dashboard。然后放开鼠标键并再次按下键盘快捷键隐藏 Dashboard。

■

禁用添加新 Widget 的波纹效果： 这个选项允许您启用或禁用在 Dashboard 中添加新 Widget 的波纹效果。

其它程序：

■

Apple 远程桌面：该选项允许您显示或隐藏 Apple 远程桌面的调试菜单。

■

iCal：此选项可在 iCal 的菜单栏显示调试菜单。

■

iChat：此选项可启用和禁用 iChat 中的数据探测器。这允许 iChat 恢复特定类型的信息并从本地菜单请求合适的操作。

■

终端：启用终端窗口聚焦可以在将鼠标移向其窗口时迅速激活。

■

Interface Builder：这个选项默认在 3.2 版中禁用，可以启用程序中的 AppleScript 检查器。

点按 "恢复默认设置" 可以取消 OnyX 或其它程序所做的更改。执行后，参数将被重置且系统和程序的显示会还原为首次安装的状态。

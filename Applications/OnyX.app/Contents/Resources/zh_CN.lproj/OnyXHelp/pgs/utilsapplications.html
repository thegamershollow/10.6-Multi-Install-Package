应用程序

应用程序面板允许您执行默认在 Mac OS X 中但用户不易访问的程序。

屏幕共享:

使用 Mac OS X v10.5，您可以连接到网络上的另一台电脑并在您的电脑上显示那台电脑的屏幕。共享电脑屏幕时，您可以控制那台电脑上发生的情况，如打开文稿或应用程序，打开、移动或关闭窗口，甚至关闭电脑。

您也可以与网络中的其他电脑共享您的屏幕。

网络诊断:

您可以使用“网络诊断”帮助您解决 Internet 连接问题。如果不能连接到 Internet，“网络诊断”将会打开并引导您解决问题。

您可以使用“网络诊断”来诊断与以太网、AirPort、内置调制解调器或外置调制解调器相关的连接问题。如果您尝试通过其他方法连接到 Internet，请参阅您的 Internet 帐户附带的文稿。

字符显示窗口:

可以使用“字符显示窗口”在文稿中输入特殊字符，如数学符号、带重音符的字母或箭头以及其他“装饰符号”。

您也可以使用“字符显示窗口”来输入日文、繁体中文、简体中文和韩文字符以及其他语言的字符。

字符调板可用于大多数应用程序，从菜单的编辑 > 特殊字符进入。

目录实用工具:

“目录实用工具”用于确定 Mac OS X 电脑可以使用的目录服务以及它连接到特定目录域的方式。 “目录实用工具”也为在特定目录域中查找鉴定和联系信息定义搜索策略。

查看目录工具帮助了解更多信息。

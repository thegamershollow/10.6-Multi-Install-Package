Cache di Sistema

Il pannello Sistema della finestra Elimina e della finestra Automatizza permette di rimuovere i files della cache nella cartella System. Qualunque file usato dal Sistema può essere rimosso senza rischi. La cancellazione di questi files permette di recuperare spazio sul disco fisso e previene la possibilità che un file corrotto possa ostacolare le funzioni del computer.

Il computer potrebbe rallentare temporaneamente dopo il completamento di questa operazione.

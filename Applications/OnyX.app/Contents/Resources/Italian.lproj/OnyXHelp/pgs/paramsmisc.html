Altro

Il pannello Altro della finestra Parametri permette di modificare le preferenze nascoste delle applicazioni.

Utility Disco:

■

Verifica le Immagini Disco (.dmg): Questa opzione consente di attivare o disattivare la verifica delle Immagini Disco quando queste vengono attivate.

■

Aggiungi altri formati di Immagini Disco: Questa opzione consente di aggiungere ulteriori formati quando desideri convertire un'immagine disco o creare un'immagine disco a partire da una cartella. Questa funzione non ha effetto nel creare un'immagine disco nuova.

■

Attiva il Menù Debug: Seleziona questa opzione se vuoi visualizzare il menù Debug nella Utility Disco.

■

Verifica il disco dopo la masterizzazione: Questa opzione permette di abilitare o disabilitare la verifica del disco dopo la msterizzazione.

Visore Aiuto:

■

Imposta finestra in modalità fluttuante: Di default, l'applicazione Visore Aiuto mostra una "finestra flottante", cioè una finestra che si può spostare rimanendo in primo piano, sopra a tutte le altre finestre. Disabilitare questa opzione permette di visualizzare questa finestra come una qualsiasi altra finestra, quindi può essere in parte o completamente nascosta dalle altre finestre. When this option is turned off, the Help Viewer icon is visible in the Dock and in the Application Switcher.

■

Mostra icone nel Dock e in Alterna applicazioni: Di default, quando si consulta l'aiuto Mac e l'aiuto della maggior parte della applicazioni, l'icona del Visore Aiuto nonn appare né nel Dock né nell'Alterna Applicazioni (Command + Tab). Questa opzione permette di visualizzarla o nasconderla.

■

Attiva il modo Debug: Questa modalità aggiunge molte informazioni nel resoconto di Sistema (/private/var/log/system.log) durante l'uso dell'applicazione Visore Aiuto.

X11:

■

Mostra messaggio quando esci: Questa opzione permette di visualizzare o nascondere il messaggio di avviso che appare di default quando si esce da qualsiasi applicazione X11.

■

All'avvio di X11, esegui: Questo menù permette di scegliere quale applicazione caricare automaticamente quando viene lanciato X11.

Condivisione schermo:

■

Mostra browser Bonjour: Questa opzione permette di visualizzare o nascondere l'elenco delle macchine disponibili per il protocollo Bonjour Apple.

■

Attiva tutti i pulsanti nella barra strumenti: Questa opzione permette di visualizzare o nascondere tutta la barra strumenti durante la condivisione schermo.

Dashboard:

■

Attiva Dashboard ed i widgets: Disattivando questa opzione, non avrà più effetto fare click sull'icona di Dashboard nel Dock o premere il tasto abbreviazione o posizionare il mouse nell'angolo attivo dello schermo configurati nelle Preferenze di Sistema.

■

Abilita la modalità Sviluppatore nel Dashboard: Questa opzione vi permette di mettere e usare le widgets del Dashboard sul Desktop del vostro Mac. Selezionate questa opzione e cliccate sul pulsante Applica (il Dock sarà riavviato). Poi, nel menù Apple, selezionate "Preferenze di Sistema...", cliccate sull'icona Dashboard e Exposé e settate la scorciatoia da tastiera di Dashboard.

- Per mettere una widget sul desktop, cliccate l'icona del Dashboard nel Dock o premete la scorciatoia da tastiera settata precedentemente per visualizzare il Dashboard. Cliccate il widget desiderato e con il pulsante del mouse premuto (come se voleste visualizzarlo) premete la scorciatoia da tastiera per nascondere il Dashboard. Il widget comparirà sul desktop e potrà essere usato come qualsiasi altra applicazione. In questo modo potete mettere molte altre widgets sul desktop del tuo Mac.

- Per rimuovere un widget dal Desktop, cliccate il widget desiderato e con il pulsante del mouse premuto premete la scorciatoia di tastiera per visualizzare il Dashboard. Poi rilasciate il pulsante del mouse e premete la scorciatoia da tastiera per nascondere il Dashboard.

■

Turn off the ripple effect when adding a new widget: You can use this option to turn on or off the ripple effect that appears when you add a new widget in Dashboard.

Altre applicazioni:

■

Apple Remote Desktop: Questa opzione permette di mostrare o nascondere un menu Debug nella barra di Apple Remote Desktop.

■

iCal: Spunta questa casella se vuoi visualizzare un menù Debug nell'applicazione iCal.

■

iChat: Questa opzione attiva o disattiva i Data Detectors in iChat. E' un meccanismo che permette ad iChat di recuperare certe informazioni e di proporre un'adeguta azione da un menù contestuale.

■

Terminale: Turning on the Terminal window Focus enables choosing which Terminal window is active by placing the mouse pointer above it.

■

Interface Builder: You can use this option, turned off by default from version 3.2, to turn on the AppleScript Inspector in the Interface Builder application.

Fate clic su Ripristina Default per eliminare tutte le modifiche apportate da OnyX o da un'altra applicazione. I parametri saranno riportati al loro valore iniziale di quando il Sistema e le applicazioni sono state installate per la prima volta.

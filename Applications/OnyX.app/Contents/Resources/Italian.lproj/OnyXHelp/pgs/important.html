Effettuate il Backup dei vostri dati:

Prima di usare OnyX, effettuate il backup dei vostri dati.

OnyX non è privo di bachi, e nessun utente è infallibile. Effettuate il backup usando Time Machine o qualsiasi altra utility di terze parti.

Verifying startup disk:

Before running Maintenance or Cleaning tasks, you should check the S.M.A.R.T. status of the startup disk and its system files.

These vérifications are essential and can be made at the OnyX launch or from the Verify pane. Errors in the structure of system files are not uncommon and can have disastrous consequences if they're not repaired.

Blocchi del Computer:

OnyX non può distruggere il Sistema.

Se il vostro computer si blocca durante la verifica del volume, non preoccupatevi. E' un comportamento previsto e normale! Una volta che OnyX ha finito di verificare il computer sarà di nuovo reattivo ai vostri comandi.

Rimozione dei dati personali:

OnyX non rimuove i vostri dati personali…

...ma dovete osservare alcune regole basilari che riguardano il file system. Mac OS X è un sistema operativo basato su Unix ed ha una ben definita struttura. I vostri dati dovranno essere immagazzinati nella directory HOME, nelle cartelle esistenti, per esempio 'Documenti', 'Immagini', 'Musica', ecc..., o in una cartella aggiuntiva che avete creato precedentemente nella directory HOME. Non dovreste mai immagazzinare i vostri documenti o applicazioni in cartelle subordinate al sistema operativo, per es. cartelle chiamate'Libreria', 'Sistema', o in nessuna delle sottocartelle di queste cartelle.

Chiunque può fare un errore, o manipolare dei dati in maniera sbagliata...

Effettuate il Backup dei dati con regolarità!

Problemi con una funzione nascosta:

OnyX non sarà sempre in grado di correggere una funzione che ha un comportamento indesiderato.

Infatti, le funzioni nella finestra Parametri sono state create dagli sviluppatori del Sistema o delle applicazioni correlate. OnyX abilita o disabilita semplicemente queste funzioni, usando un semplice comando, a vostra discrezione. Se qualcuna di queste funzioni provoca un problema, disabilitatela. Non dimenticate: se una funzione è nascosta, deve esserci una buona ragione...

Aggiornamenti:

Aggiornate sempre OnyX e il vostro Sistema.

Di solito l'ultima versione corregge generalmente almeno una parte dei problemi presenti nella versione precedente.

Uninstall OnyX:

To uninstall OnyX, go in the OnyX menu and choose Uninstall OnyX. In the confirmation message that appears, click the Continue button. The application and the components created while installing and using OnyX will be removed.

Per annullare la visualizzazione di questa finestra di avviso all'apertura di OnyX, aprite le Preferenze > "Messaggi e resoconti", e deselezionate l'opzione "Mostra le informazioni importanti".

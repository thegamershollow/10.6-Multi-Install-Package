Applications

You can use the Applications pane to execute applications present by default in Mac OS X but not easily accessible to you.

condivisione dello schermo:

In Mac OS X, puoi connetterti a un altro computer sul network e visualizzare lo schermo del computer sul tuo computer. Durante la condivisione dello schermo, puoi controllare tutto quello che accade sull'altro computer, ad esempio aprire documenti o applicazioni, aprire, spostare o chiudere finestre e persino spegnere il computer.

Inoltre, puoi condividere lo schermo del tuo computer con un altro computer del network.

Diagnosi Network:

Puoi usare Diagnosi Network per risolvere i problemi di connessione a Internet. Se non riesci a stabilire una connessione a Internet, Diagnosi Network si aprirà automaticamente e ti guiderà per risolvere il problema.

Puoi servirti di Diagnosi Network per analizzare i problemi legati alle connessioni con Ethernet, AirPort o modem interno o esterno. Se stai provando a connetterti a Internet con un altro metodo, consulta la documentazione fornita con l'account Internet.

Visore Caratteri:

Utilizza Visore Caratteri per inserire nei documenti caratteri speciali e simboli, ad esempio simboli matematici, lettere accentate, frecce e altri segni grafici.

Puoi utilizzare Visore Caratteri anche per inserire caratteri della lingua giapponese, cinese, tradizionale e semplificato, coreano e caratteri di altre lingue.

The Character palette is available in most applications from the Special Characters command in the Edit menu.

Utility Directory:

Utility Directory determina i servizi directory utilizzati da un computer Mac OS X e la modalità di connessione a domini di directory specifici. Utility Directory, inoltre, definisce i criteri di ricerca per trovare le informazioni sui contatti e sull'autenticazione in domini di directory specifici.

See the Directory Utility Help for more info.

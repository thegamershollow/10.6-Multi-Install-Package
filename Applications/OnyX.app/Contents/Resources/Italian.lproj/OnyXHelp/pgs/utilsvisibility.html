Visibility

You can use the Visibility pane to act on the visibility state of a volume, folder, or file.

To modify the visibility of a volume:

Select the volume in the local menu, then click the Show button (to make it visible) or the Hide button (to make it invisible).

To modify the visibility of a folder:

Click the Show Folder button to make it visible or Hide Folder to make it invisible, then in the dialog that appears, select the folder.

Note: These two buttons can possibly influence the visibility of a volume.

To modify the visibility of a file:

Click the Show File button to make it visible or Hide File to make it invisible, then in the dialog that appears, select the file.

Warning: All of these operations relaunch the Finder. Verify that no operation is in progress (copying, moving, burning, and emptying the Trash).

Scripts di manutenzione

Mac OS X include un task manager (launchd) il quale automatizza i processi di pulizia del Sistema eseguendo le utility "periodicamente". Queste utility lanciano tre scripts BSD standard che eseguono molte operazioni di manutenzione.

Lanciare gli scripts di manutenzione:

Aprite la finestra "Manutenzione" e cliccate sul pannello "Scripts".

Selezionate lo script che si intende eseguire (giornaliero, settimanale, e/o mensile).

Cliccate sul pulsante "esegui".

Nota: La finestra Automatizza permette di lanciare tutti gli scripts contemporaneamente.

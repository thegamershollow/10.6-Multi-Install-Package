Journaling

Questa opzione permette di abilitare o disabilitare l'aggiornamento del disco fisso.

Se il "journaling" di un disco è attivo, Mac OS X conserva una registrazione permanente di tutti i cambiamenti fatti ai files sul disco. Se il vostro computer si spegne a causa di una mancanza di corrente o per qualche altro motivo, Mac OS X usa la registrazione per ripristinare il disco fisso alle ultime condizioni accettabili prima dello spegnimento.

Nota Bene: Se il "journaling" è disattivo, non è possible di verificare la struttura del disco di avvio.

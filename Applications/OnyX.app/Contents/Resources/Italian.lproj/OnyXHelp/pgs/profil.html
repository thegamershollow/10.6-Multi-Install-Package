Profilo

Il pannello Profilo della finestra Info permette di visualizzare o salvare il profilo della tua configurazione hardware e software.

Esistono 3 tipi di profilo:

- Profilo minimo: Questo profilo contiene solo un minimo di informazioni e omette i codici identificativi del vostro computer (numero seriale, indirizzo IP...).

- Profilo di base: Questo profilo contiene la maggior parte delle informazioni hardware e omette la maggior parte delle informazioni software.

- Profilo completo: Questo profilo contiene tutte le informazioni della configurazione hardware e software.

Ogni profilo può essere salvato in formato Testo (file .txt leggibili con TextEdit o qualsiasi altro editor di testo) o in formato XML (file .spx leggibile dall'applicazione System Profiler).

Per visualizzare un profilo:

Selezionate il profilo che volete visualizzare dal menù "Stile profilo".

Cliccate il pulsante "Mostra".

Per salvare un profilo:

Selezionate il profilo che volete salvare dal menù"Stile profilo".

Selezionate il formato desiderato(Testo o XML) dal menù "Formato".

Cliccate sul pulsante "Salva". Il file del profilo sarà salvato sulla vostra Scrivania.

Fehler beim Überprüfen des Startlaufwerks

Wenn OnyX während der Überprüfung der Dateistruktur einen Fehler entdeckt, müssen Sie dies so schnell wie möglich korrigieren.

To repair a disk:

Beenden Sie OnyX.

Legen Sie die Mac OS X-Installations-CD ein und starten Sie den Computer neu. Halten Sie dabei die Taste "C" gedrückt.

Wählen Sie Ihre Sprache.

Wählen Sie im Menü "Dienstprogramme" das Festplatten-Dienstprogramm.

Wählen Sie ihr Startlaufwerk in der linken Spalte aus.

Klicken Sie im Tab "Erste Hilfe" den Button "Reparieren" und warten Sie.

Starten Sie den Computer neu, wie üblich.

Meist sind diese Fehler nicht schwer wiegend und können vom Festplatten-Dienstprogramm behoben werden. Trotzdem könnte es nötig werden, dass Sie das Wartungsprogramm eines anderen Herstellers einsetzen müssen, z.B. Drive Genius, DiskWarrior oder TechToolPro.

Applications

You can use the Applications pane to execute applications present by default in Mac OS X but not easily accessible to you.

Bildschirmfreigabe:

Mit Mac OS X können Sie eine Verbindung zu einem anderen Computer in Ihrem Netzwerk herstellen und die Bildschirmanzeige dieses Computers auf Ihrem Bildschirm anzeigen. Während Sie auf den Bildschirm dieses Computers zugreifen, können Sie die Vorgänge auf diesem Computer steuern. So ist es beispielsweise möglich, per Fernzugriff Dokumente oder Programme zu öffnen, Fenster zu öffnen, zu bewegen oder zu schließen und sogar den Computer auszuschalten.

Sie können auch den Bildschirm Ihres Computers für andere Computer im Netzwerk freigeben.

Netzwerkdiagnose:

Mithilfe des Programms "Netzwerkdiagnose" können Sie Probleme beim Herstellen einer Internetverbindung lösen. Wenn Sie keine Internetverbindung herstellen können, wird das Programm "Netzwerkdiagnose" geöffnet und eine Anleitung zur schrittweisen Lösung des Problems angezeigt.

Durch Verwendung der Netzwerkdiagnose können Sie Probleme mit Ihren Verbindungen über Ethernet, AirPort oder ein internes bzw. externes Modem ermitteln. Wenn Sie versuchen, mit einer anderen Methode eine Internetverbindung herzustellen, lesen Sie die Dokumentation zu Ihrem Internet-Account.

Zeichenübersicht:

Mithilfe der Zeichenübersicht können Sie Sonderzeichen und Symbole, wie z. B. mathematische Symbole, Buchstaben mit Akzenten oder Pfeile und andere grafische Sonderzeichen, in Ihre Dokumente eingeben.

Mit dieser Übersicht können Sie auch Zeichen aus anderen Sprachen eingeben, darunter Japanisch, Traditionelles Chinesisch, Vereinfachtes Chinesisch und Koreanisch.

The Character palette is available in most applications from the Special Characters command in the Edit menu.

Verzeichnisdienste:

Das Programm "Verzeichnisdienste" legt fest, welche Verzeichnisdienste (Directory Services) von einem Mac OS X-Computer verwendet werden und wie dieser die Verbindung zu bestimmten Verzeichnis-Domains herstellt. Das Programm "Verzeichnisdienste" definiert außerdem Suchpfade für die Suche nach Identifikations- und Kontaktinformationen in bestimmten Verzeichnis-Domains.

See the Directory Utility Help for more info.

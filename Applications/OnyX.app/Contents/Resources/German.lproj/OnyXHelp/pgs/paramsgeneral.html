Allgemein

The General tab allows adjustment of some global parameters.

Fenster:

■

Position der Rollpfeile: You can use OnyX to choose among four types of arrows in window scroll bars. Select one of these options to modify the parameters immediately and relaunch the Finder; the applications already in memory must be relaunched to show the changes.

■

Geschwindigkeit für Größenänderung der Fenster: You can use this option to set the speed that a window's sheet is displayed. A "sheet" is a modal dialog which is attached to a window and asks for input from the user.

CrashReporter:

■

Modus für den CrashReporter: By default, when an application quits unexpectedly, an alert message appears that allows you to reopen the application and send a report to Apple. It's the basic mode of operation of Mac OS X, but two other modes exist. The Developer mode displays an alert message with miscellaneous debug information. The Server mode, turned on by default in Mac OS X Server, doesn’t display any message but stores the Crash reports in the Library/Logs/CrashReporter and Library/Logs/CrashReporter folder in your home folder.

Netzwerk-Volumes:

■

Erzeugung von .DS_Store-Dateien ausschalten: You can use this option to prevent .DS_Store files creation on network volumes. By default, these files are invisible under Mac OS X, but they become visible (and they are useless) on other operating systems.

Bildschirmfoto:

■

Bildschirmfoto sichern als: You can use this menu to change the format (PNG by default) of Screen Captures made by using the keyboard shortcuts. You may choose among nine formats: BMP, GIF, JPEG, JPEG 2000, OpenEXR, PDF, PNG, TGA, and TIFF.

■

Schatten in Bildschirmfotos: By default, when you capture a window (Shift + Command + 4 then Space bar), the shadow of this one is also captured. You can use this option to exclude the shadow of window captures.

■

Pfad: This option specifies the saving-path for screenshots (your Desktop, by default). To change this, click the Select button, choose the folder, and click the Choose button.

■

Name: You can choose the name of the screen captures by typing the name in the Name field and clicking the Apply button.

Benutzte Dokumente:

■

Anzeige des Menüpunkts "Benutzte Dokumente" (Menü Ablage) in Programmen einschalten: The majority of the applications that allow you to open files have an Open Recent menu item in their File menu. By default, this submenu displays a list of the ten last documents opened by the application. You can use this function to turn the appearance of this list on or off. However, certain applications (Photoshop, GoLive, GraphicConverter, and so on) use their own routine to store Recent Items.

■

Anzahl benutzter Einträge: You can use this field to set the number of entries in the Open Recent command of the File menu of applications that allow it.

■

Anzahl der zuletzt besuchten Orte in den Dialogen: This local menu allows you to display between 1 and 12 recent places (the default is 5) in the Open/Save dialogs.

Animierter Schreibtisch-Hintergrund:

■

Animierter Schreibtisch-Hintergrund: You can use this option to choose a Screen Saver among those installed by Mac OS X and start it as a Desktop Background.

Downloaded applications:

■

Die Warnmeldung beim ersten Start von heruntergeladenen Programmen deaktivieren: At first launch of an application downloaded from the Internet, an alert message appears. This message gives miscellaneous information on the source of the downloaded file, and you can open the file. You can use this option to show or hide this alert message.

Preferences Panes:

■

Archive: This button launches the installation of an Archives Preferences pane that allows changing the settings of the Archive Utility application.

■

Prozessor: This button launches the installation of a Processor Preferences pane.

To uninstall a Preferences pane, click its icon with the right button of the mouse or hold the Control key while clicking the icon with the left mouse button.

Klicken Sie auf den "Standard"-Button um alle von OnyX oder einem anderen Programm gemachten Einstellungen zurück zu setzen. Danach sind alle Anzeigen und Einstellungen so, als wäre das System eben installiert worden.

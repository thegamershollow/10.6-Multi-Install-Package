Safari

You can use the Safari tab of the Parameters pane to set the parameters of some hidden Safari functions.

Navigation:

■

Automatisch PDF-Seiten in Safari anzeigen: By default, Safari opens and displays the content of PDF files found on the web. Nevertheless, you can disable this behavior. PDF files will be downloaded.

■

Turn off Safari's image cache: When you browse the Internet, Safari creates two images of the contents of each webpage viewed (a JPEG and another PNG). These images are used as seen when viewing the Top Sites or History. You can use this option to turn on or turn off the creation of these images.

■

Vor-/Zurück-Buttons aktivieren: You can use this option to turn on and off the Back and Forward buttons located in Safari's address bar as well as the History menu which appears when the user clicks on these buttons.

■

Ein-Fenster-Betrieb aktivieren: Some webpages have links opening a new window. You can use this option to open these pages in a new tab.

■

Schließen mehrerer Seiten bestätigen lassen: You can turn on or off the confirmation message which appears when closing multiple windows or a window containing multiple tabs.

Verschiedene Optionen:

■

Safari-Develop-Menü anzeigen: Safari has hidden features that can be turned on or off with this option. You can use this menu to open the current webpage in another browser, change the User Agent, display the Web Inspector, turn on or turn off some functions, and so on. This option also adds An Inspect Element item in the shortcut menu of Safari. By default, the Web Inspector appears in an individual window, but it can be merged with the displayed page by clicking the icon intended for this purpose located at the Inspector's bottom left.

■

Lesezeichen-Icons anzeigen: By default, Safari saves the Bookmark icons of all websites visited. Nevertheless, you can choose not to display these icons by deselecting this option.

■

In den Tooltipps die URL von Links anzeigen: You can turn on this function to display the complete address of links (webpage and email) in a contextual ToolTip when the mouse hovers above these links.

Drucken:

■

Seitenhintergrund: Select this option if you want to print the background of webpages with Safari.

■

Kopf- und Fußzeilen: You can use this option to decide whether you want to print the headers and the footers of webpages.

You can also modify this parameter directly in Safari. (Choose Print in the File menu, then select the "Print backgrounds" and "Print headers and footers" options.)

Logs:

Enter the values you want, then click the Apply button.

■

Maximale im Menü "Verlauf" sichtbare Anzahl Seiten: You can use this field to set the number of items appearing in the History menu in Safari.

■

Anzahl der Tage: You can use this field to set the number of days preserved in the History of Safari.

Zoom Text Only:

This option in Safari, accessible from the View menu, is turned off as a hidden feature of WebKit. You can use the "Turn on option" button to remove the offending property and fix the bug.

Klicken Sie auf den "Standard"-Button um alle von OnyX oder einem anderen Programm gemachten Einstellungen zurück zu setzen. Danach sind alle Anzeigen und Einstellungen so, als wäre das System eben installiert worden.

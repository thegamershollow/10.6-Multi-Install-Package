Benutzer

The User tab of the Cleaning pane and the Automation pane allows you to remove any Cache files (Applications and Preferences Caches, Doc and so on) and Temporary items. The deletion of these files recovers some disk space and prevents a corrupt file from impacting computer functions.

You should run this command occasionally. A slight temporary decrease in speed will be observed (when applications are open, displaying System Preferences, searching for updates, and so on).

The cleaning of User Cache from the Automation pane doesn't remove the Audio Units Cache.

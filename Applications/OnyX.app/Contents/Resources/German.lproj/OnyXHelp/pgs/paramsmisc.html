Verschiedenes

You can use the Misc. tab of the Parameters pane to set parameters for the hidden functions of the applications.

Festplatten-Dienstprogramm:

■

Überprüfen von Disk-Images (.dmg): You can use this option to turn on or off the verification when you mount Disk Images.

■

Weitere Formate für Disk-Images anfügen: You can use this option to add supplementary formats if you wish to convert a disk image or create a disk image from a folder. You cannot use this function to create a new empty disk image.

■

Debug-Menü anzeigen: Check this checkbox if you want to display a Debug menu in the Disk Utility menu bar.

■

Medium nach dem Brennen überprüfen: You can use this option to turn on or off Verification after burning a disk.

Help Viewer:

■

Schwebemodus für Fenster aktivieren: By default, the Help Viewer application displays a "floating window". (That is a window that floats in the foreground, above all other open windows.) Turn off this option to allow displaying this window like any other window, which can be partly or entirely hidden by other windows. When this option is turned off, the Help Viewer icon is visible in the Dock and in the Application Switcher.

■

Das Icon im Dock und im Programmumschalter zeigen: By default, when consulting the Mac Help and the Help of most applications, the Help Viewer icon appears neither in the Dock nor in the the Application Switcher (Command + Tab). You can use this option to display or hide it.

■

Debug-Modus aktivieren: This mode adds much information in the System log (/private/var/log/system.log) while using the Help Viewer application.

X11:

■

Warnhinweis vor dem Benden des Programms: You can use this option to show or hide the alert message that appears by default when quitting any X11 application.

■

Beim Start von X11 ausführen: You can use this menu to choose the application which will load automatically when you launch X11.

Screen Sharing:

■

Bonjour-Browser zeigen: You can use this option to show or hide the list of computers available to the Apple Bonjour protocol.

■

Vollständige Werkzeugleiste zeigen: You can use this option to show or hide the full toolbar while screen sharing.

Dashboard:

■

Dashboard und Widgets aktivieren: If you turn on this option, clicking on the Dashboard icon in the Dock will display widgets.

■

Entwickler-Modus von Dashboard aktivieren: Mit dieser Option können Sie Widgets wie Programme vom Schreibtisch aus starten. Select this option, and click the Apply button. (The Dock will be reloaded.) Then, in the Apple menu, select "System Preferences...", click the Dashboard and Exposé icon, and set the Dashboard's Keyboard Shortcut.

- To place a widget on the Desktop, click the Dashboard icon in the Dock or press the Keyboard Shortcut you previously set to display Dashboard. Click the desired widget, and hold the mouse-button (as if you wanted to remove it), then press the Keyboard Shortcut to hide Dashboard. The widget will remain on the Desktop and can be used as any other application. You can place as many widgets as you want on the Desktop of your Mac.

- To remove a widget from the Desktop, click and hold the mouse button, then press the Keyboard Shortcut to display Dashboard. Then, release the mouse button, and press the Keyboard Shortcut to hide the Dashboard.

■

Turn off the ripple effect when adding a new widget: You can use this option to turn on or off the ripple effect that appears when you add a new widget in Dashboard.

Other applications:

■

Apple Remote Desktop: This option shows or hides a Debug menu in the Apple Remote Desktop menu bar.

■

iCal: This option displays a Debug menu in the iCal menu bar.

■

iChat: This option turns on and off Data Detectors in iChat. It is a mechanism that allows iChat to recover certain types of information and to propose a suitable action from a local menu.

■

Terminal: Turning on the Terminal window Focus enables choosing which Terminal window is active by placing the mouse pointer above it.

■

Interface Builder: You can use this option, turned off by default from version 3.2, to turn on the AppleScript Inspector in the Interface Builder application.

Klicken Sie auf den "Standard"-Button um alle von OnyX oder einem anderen Programm gemachten Einstellungen zurück zu setzen. Danach sind alle Anzeigen und Einstellungen so, als wäre das System eben installiert worden.

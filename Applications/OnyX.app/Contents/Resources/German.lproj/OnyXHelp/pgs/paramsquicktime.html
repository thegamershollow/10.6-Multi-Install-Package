QuickTime

You can use the QuickTime tab in the Parameters pane to set the parameters of some QuickTime hidden functions.

Playback controls and title bar:

■

Always hide: After starting to play music and when the mouse leaves the windows, the Playback controls and title bar will be automatically hidden.

■

Never display the title bar: This option completely hides the title bar of the windows of QuickTime Player.

WARNING: If the title bar is not visible anymore, it's impossible to close, minimize, and zoom windows.

Misc. options:

■

Autoplay movies: Select this option if you want to automatically play movies after opening them.

■

Corners of the windows rounded: You can use this option to choose between contours of windows rounded or with right angles.

■

Keep in Full Screen mode: By default, when you view a movie in full screen mode and then you rock toward another application, the window resumes its real size. You can use this option to keep the full screen mode even if QuickTime Player isn't in the foreground anymore.

QuickTime Player 7:

■

Filminformationen des QuickTime-Players beim Start anzeigen: You can use this option to automatically display the QuickTime Player 7's Info window at launch or when you open a file. This doesn't work with QuickTime Player X.

Klicken Sie auf den "Standard"-Button um alle von OnyX oder einem anderen Programm gemachten Einstellungen zurück zu setzen. Danach sind alle Anzeigen und Einstellungen so, als wäre das System eben installiert worden.

Login

La pestanya Login de la consola de Paràmetres li permet configurar els paràmetres per a diverses opcions de la finestra de validació.

Background image:

You can change the background image of the login window.

■

Background: You can use the Select button to choose a replacement image for the background of the login window. By default, the application directs you to the folder containing the Desktop Pictures of Mac OS X (/Library/Desktop Pictures/...). You can choose another image, but it must be accessible by all users, for example in the Public, Shared, or Sites folder.
Formats accepted: BMP, GIF, JPEG, JPEG 2000, PNG, PSD, TGA, and TIFF.

■

Restore: You can use this option to put the original background image in the login window.

Missatge:

■

Visualitzar missatge a la finestra de validació: Aquesta opció li permet mostrar un missatge de text a la finestra de validació. Seleccioneu aquesta opció, introduïu el text desitjat al camp inferior i premi el botó Aplica. Avís: Aquesta funció no accepta alguns caràcters especials.

Altres Opcions:

■

Desactivar les ordres Reiniciar, Apagar equip i Tancar sessió del menú Poma: Aquesta opció us permet activar o desactivar el mode Kiosk. En activar aquesta manera es desactiven les ordres "Reiniciar", "Apagar equip" i "Tancar Sessió..." al menú Poma. Qualsevol modificació del mode Kiosk requereix el reinici de l'ordinador.
Important: Utilitza aquesta opció només si els administradors es mostren a la llista d'usuaris.

■

Ocultar les ordres Repos, Reiniciar i Apagar equip a la finestra de validació: Aquesta opció oculta els tres botons "Repòs", "Reiniciar" i "Apagar" ubicats a la part inferior de la finestra de validació.

■

Mostrar info a la finestra de validació: Per defecte, el nom de l'equip apareix a la part de dalt de la finestra de validació. Aquest menú us permet seleccionar la visualització d'altres informacions (Versió del sistema o compilació, número de sèrie, adreça IP, Estat dels Serveis de Directori o data i hora).

■

Visualitzar a la llista d'usuaris: Aquesta opció li permet mostrar o amagar els usuaris (Administradors, locals i de Xarxa...) en la llista d'usuaris en la Finestra de validació.
Avís: Si està actiu el mode Kiosk, no tregui la selecció als administradors en la llista d'usuaris.

Mode d'Arrencada:

Aquest menú us permet configurar la manera d'arrencada de l'ordinador.

■

Normal: El mode Normal mostra un logo d'Apple en gris fosc (la poma) en un fons gris clar, mostra missatges d'estat sobre una barra de progrés aqua mentre es van executant diverses inicialitzacions, i a continuació mostra l'escriptori al Finder.

■

Mode Segur: El Mode Segur és l'estat en què Mac OS X queda després d'un Arrencada segura. En arrencar en Mode Segur es força una comprovació de l'estat del disc d'arrencada, carregant només les extensions del nucli necessàries, executa únicament els elements d'arrencada instal·lats per Apple, desactiva totes les fonts no instal·lades pel sistema, mou totes les memòries cau de fonts a la Paperera i desactiva qualsevol element de Validació. El Mode Segur pot ser útil per a la resolució de problemes. A més, algunes característiques de Mac OS X no funcionen quan s'ha arrencat en Mode Segur. Per exemple, no pot utilitzar el Reproductor de DVD, capturar vídeo en iMovie, utilitzar una targeta AirPort, utilitzar dispositius d'entrada o sortida d'àudio, o utilitzar un mòdem USB intern o extern.

■

Detallat: El mode Detallat (Verbose) visualitza una pantalla negra amb text en blanc que mostra els detalls del procés d'arrencada línia a línia, mostrant a continuació l'Escriptori al Finder.

■

Usuari únic: El mode Usuari únic visualitza una pantalla negra amb text en blanc que mostra els detalls del procés d'arrencada línia a línia, mostrant a continuació un símbol d'ordres; per tornar al mode Normal, introduïu l'ordre nvram boot-args = "" però si a vostè no li agrada el terminal, oblidi's del mode d'usuari únic.

Premeu a Restablir valors per defecte per a cancel·lar totes les modificacions fetes amb OnyX o per qualsevol altra aplicació. Després d'executar-lo, els paràmetres seran reiniciats i mostrats tal com es van definir a la primera instal·lació del sistema i les aplicacions.

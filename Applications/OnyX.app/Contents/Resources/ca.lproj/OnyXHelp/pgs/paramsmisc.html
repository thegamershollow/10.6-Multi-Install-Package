Altres

La pestanya Altres de la consola de Paràmetres li permet configurar paràmetres de funcions ocultes d'aplicacions.

Utilitat de Disc:

■

Verificar Imatges de disc (.dmg): Aquesta opció us permet activar o desactivar la verificació quan munta Imatges de Disc.

■

Afegir altres formats d'Imatges de Disc: Aquesta opció us permet afegir formats suplementaris quan vulgui convertir una imatge de disc o crear una imatge de disc des d'una carpeta. Aquesta funció no és operativa a l'hora de crear una nova imatge de disc buida.

■

Activar el menú Debug: Seleccioneu aquesta opció si voleu mostrar el menú Debug a la barra de menús de la Utilitat de Discos.

■

Verificar els discs després de gravar: Aquesta opció us permet activar o desactivar la verificació després de gravar un disc.

Visor d'Ajuda:

■

Configurar finestres en mode flotant: By default, the Help Viewer application displays a "floating window". (That is a window that floats in the foreground, above all other open windows.) Turn off this option to allow displaying this window like any other window, which can be partly or entirely hidden by other windows. When this option is turned off, the Help Viewer icon is visible in the Dock and in the Application Switcher.

■

Mostra la icona al Dock i al commutador d'aplicacions: Per defecte, quan es consulta l'ajuda de Mac i de moltes de les aplicacions, la icona del Visor d'ajuda no apareix ni en el Dock ni al commutador d'aplicacions (Comando + Tab). Aquesta opció li permet mostrar-la o ocultar-la.

■

Activar el mode Debug: Aquest mode afegeix diversa informació en el registre de sistema (/private/var/log/system.log) mentre utilitza l'aplicació del Visor d'Ajuda.

X11:

■

Mostrar missatge d'avís abans de sortir de l'aplicació: Aquesta opció li permet mostrar o amagar el missatge d'avís que apareix per defecte en sortir de l'aplicació X11.

■

A l'inici de X11, executar: Aquest menú us permet seleccionar l'aplicació que voleu executar automàticament en iniciar X11.

Compartir Pantalla:

■

Mostrar navegador Bonjour: Aquesta opció li permet mostrar o amagar la llista d'equips disponible mitjançant el protocol Bonjour d'Apple.

■

Mostrar barra d'eines completa: Aquesta opció li permet mostrar o ocultar la barra d'eines completa mentre comparteix la pantalla.

Dashboard:

■

Activar Dashboard i els seus Widgets: Si activa aquesta opció, en prémer sobre la icona del Dashboard al Dock es mostraran els widgets.

■

Activar el Mode Desenvolupador del Dashboard: Aquesta opció li permet situar i usar els widgets del Dashboard a l'escriptori del seu Mac. Seleccioneu aquesta opció i premeu el botó Aplicar (el Dock serà reiniciat). A continuació, al menú Poma, seleccioneu "Preferències del sistema...", premi sobre la icona Dashboard i Exposé i configuri les Dreceres de Teclat del Dashboard.

- Per posar un widget a l'escriptori, premi sobre la icona Dashboard al Dock o premi la drecera de teclat prèviament configurada per mostrar el Dashboard. Premi sobre el widget desitjat i amb el botó del ratolí premut (com si el volgués eliminar) premi la drecera de teclat per ocultar el Dashboard. El widget romandrà en l'escriptori podent utilitzar-se com qualsevol altra aplicació. D'aquesta manera, vostè pot posar diversos widgets a l'escriptori del seu Mac.

- Per eliminar un widget de l'escriptori, feu clic sobre el widget desitjat i, amb el botó del ratolí premut, premeu la drecera de teclat per mostrar el Dashboard. A continuació deixeu anar el botó del ratolí i premeu la drecera de teclat per ocultar el Dashboard.

■

Turn off the ripple effect when adding a new widget: You can use this option to turn on or off the ripple effect that appears when you add a new widget in Dashboard.

Altres aplicacions:

■

Apple Remote Desktop: Aquesta opció li permet mostrar o amagar el menú Debug a la barra de menús d'Apple Remote Desktop.

■

iCal: Aquesta opció mostra el menú Debug a la barra de menús de iCal.

■

iChat: Aquesta opció activa i desactiva els Detectors de Dades en iChat. Aquest és un mecanisme que permet a iChat recuperar alguns tipus d'informació i proposar una acció adaptada des d'un menú local.

■

Terminal: Activa el Focus de finestra al Terminal activant la selecció de finestra de terminal quan es passa el punter del ratolí sobre ella.

■

Interface Builder: Aquesta opció, desactivada per defecte des de la versió 3.2, li permet activar el inspector AppleScript en l'aplicació Interface Builder.

Premeu a Restablir valors per defecte per a cancel·lar totes les modificacions fetes amb OnyX o per qualsevol altra aplicació. Després d'executar-lo, els paràmetres seran reiniciats i mostrats tal com es van definir a la primera instal·lació del sistema i les aplicacions.

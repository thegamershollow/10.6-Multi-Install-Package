Scripts de Manteniment

Mac OS X inclou un gestor de tasques (launchd) que automatitza la neteja del sistema executant una utilitat de forma periòdica. Aquesta utilitat executar tres scripts BSD estàndard que efectuen nombroses operacions de manteniment.

Per executar els scripts de Manteniment:

Obrir el panell "Manteniment" i prémer la pestanya "Scripts de Manteniment".

Seleccionar els scripts que voleu executar (diari, setmanal i/o mensual).

Premeu el botó "Executar".

Nota: El panell Automatització també us permet executar els scripts de manteniment.

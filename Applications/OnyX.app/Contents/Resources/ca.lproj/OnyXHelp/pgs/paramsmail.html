Mail

La Pestanya Mail de la consola de Paràmetres li permet configurar els paràmetres de les Funcions Ocultes de Mail.

Mail:

■

Mostrar correus en TEXT enlloc de HTML quan siga possible: Per defecte, Mail mostra els nous missatges amb format RTF o HTML. Aquesta opció li permet mostrar-los en format Text, sense estil i sense la previsualització dels adjunts. Això accelera la visualització. Per veure els adjunts, premeu el botó Desa situat a la capçalera del missatge.

■

Activar Detectors de Dades: Aquesta opció activa i desactiva els Detectors de Dades en Mail. Aquest és un mecanisme que permet a un amic recuperar alguns tipus d'informació i proposar una acció adaptada des d'un menú local.

■

Turn on display attachments in messages: By default, some attachments, including images, are visible directly in the message body. You can use this option to view or not these attachments in messages.

■

Minimum font size: By default, Mail's messages appear with a minimum font size of 10. You can use this option to increase or decrease this size.

■

Amaga el nom quan es copia una adreça de correu des d'un menú de context: Per defecte, quan vostè còpia un correu des d'un element de menú contextual "Copiar Adreça", l'adreça està precedida pel nom del destinatari. Aquesta opció li permet mostrar o amagar el nom.

Premeu a Restablir valors per defecte per a cancel·lar totes les modificacions fetes amb OnyX o per qualsevol altra aplicació. Després d'executar-lo, els paràmetres seran reiniciats i mostrats tal com es van definir a la primera instal·lació del sistema i les aplicacions.

Internet

Quan vostè va a la Web, vostè descarrega, omple formularis... molts fitxers i carpetes són creats. OnyX li permet eliminar aquests elements que ocupen molt d'espai de disc.

■

Memòria cau de Navegadors: Aquesta opció esborra els fitxers creats en carregar les pàgines d'Internet en el navegador Web.

■

Memòria cau de Descàrregues: Aquesta opció esborra els fitxers creats quan els fitxers són descarregats de la Web.

■

Historial de Navegadors: Aquesta opció esborra l'historial de les pàgines que els navegadors Web fan servir per carregar de forma més ràpida.

■

Cerques recents: Aquesta opció li permet eliminar cerques recents que els navegadors Web emmagatzemen.

■

Previsualització de Pàgines Web: Quan navega per la Web, Safari crea una vista prèvia de cada pàgina com una imatge visible a la pàgina de Top Sites, els favorits i l'historial. Aquesta opció li permet eliminar totes aquestes imatges. Quan vostè reinicia Safari, un missatge li proposarà afegir vistes prèvies de les pàgines Web que hagi visitat més recentment.

■

Icones de favorits: Molts llocs Web defineixen una icona (també anomenat "favicon"), que mostra a la barra d'adreces, en els Marcadors i en les direccions del menú Historial. Aquesta opció esborra aquestes icones.

■

Valors de Formularis: La informació que vostè introdueix en els formularis dels llocs Web es guarda en una memòria cau. Aquesta opció esborra aquesta memòria cau.

■

Cookies: Molts llocs Web emmagatzemen petits fitxers, denominats "cookies", en el seu ordinador, per facilitar el retorn de les visites, per exemple, una transacció entre vostè i un comerç en línia, per evitar haver d'introduir el seu nom i contrasenya quan torni al fòrum, etc. Aquesta opció esborra aquestes cookies.

Els valors de Formularis i les galetes (poden) evitar-li el haver d'introduir noms d'usuari, contrasenyes i altra informació requerida quan torni a un formulari Web (com fòrums, botigues en línia, etc.). Usi'l amb precaució!

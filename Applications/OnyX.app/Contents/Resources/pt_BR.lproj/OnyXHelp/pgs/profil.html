Perfil

Use essa aba no painel Info para visualizar e salvar perfis de configuração de hardware e software.

Existem 3 tipos de perfis:

- Resumido: Esse perfil contém informações mínimas, omitindo informações sobre a identificação do seu computador (número de série, endereço IP, etc).

- Básico: Esse perfil contém a maioria das informações sobre o hardware, omitindo grande parte sobre o software.

- Completo: Esse perfil contém todas as informações sobre as configurações de hardware e software.

Todos os perfis podem ser salvos em formato Texto (arquivo .txt, podendo ser aberto pelo Editor de Texto e outros editores) ou em formato XML (arquivo .spx, podendo ser aberto pelo aplicativo Visão do Sistema).

Para visualizar um perfil:

Selecione o perfil desejado no menu "Estilo de Perfil".

Clique o botão "Mostrar".

Para salvar um perfil:

Selecione o perfil desejado no menu "Estilo de Perfil".

Selecione o formato no menu de mesmo nome.

Clique o botão "Salvar". O Perfil será salvo na Mesa.

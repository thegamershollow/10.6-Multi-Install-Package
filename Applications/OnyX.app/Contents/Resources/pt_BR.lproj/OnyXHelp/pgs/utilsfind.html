Busca / Banco de Dados locate

locate é um comando Unix que permite achar pastas e arquivos rapidamente ao buscar no banco de dados locate (um índice de todos os itens no disco). Ao instalar o sistema, esse banco de dados não existe e você precisa criá-lo e atualizá-lo regularmente caso pretenda usar esse comando.


Utilitários > Painel Busca:

Use a aba Busca no painel Utilitário para criar, remover e atualizar o banco de dados locate. Após a criação do banco de dados, você poderá pesquisar por itens e ao clicar em um deles na lista, o mesmo será mostrado no Finder. Esse painel mostra ainda o tamanho do banco de dados e a última vez em que foi atualizado.

Comando Reconstruir no painel Manutenção:

Caso o banco de dados locate já tenha sido criado, é possível atualizá-lo a partir do painel Manutenção, na aba Reconstruir.

Aviso: Criar e reconstruir o banco de dados locate pode levar tempo. Aguarde o término da tarefa!

Cache de Fontes

A aba Fontes, no painel Limpeza, ou o painel Automação, permitem remover os arquivos de Cache de Fontes.

Esses arquivos são usados para acelerar o carregamento das fontes. Apagá-los tornará a visualização das fontes mais lenta temporariamente, mas corrigirá eventuais problemas. Essa ação não apagará qualquer coleção pessoal de fontes.

■

A partir da aba Fontes, no painel Limpeza, é possível remover de forma independente o Cache de Fontes do Sistema, do Usuário, de aplicativos Adobe, Microsoft e Quark e de alguns aplicativos de código aberto.

■

A partir do painel Automação, todos os Caches de Fontes são eliminados.

Obs: Após apagar o Cache de Fontes de alguns aplicativos, a abertura inicial pode ser especialmente lenta.

Scripts de Manutenção

O Mac OS X inclui um gerenciador de tarefas (launchd) que executa automaticamente a limpeza do Sistema através do utilitário "periodic". Este utilitário roda três scripts padrões do BSD que executam várias tarefas de manutenção.

Para executar os Scripts:

Abra o painel Manutenção, e clique na aba Scripts.

Selecione os scripts a serem executados (diário, semanal e/ou mensal).

Clique o botão Executar.

Obs: É possível utilizar o painel Automação para executar todos os scripts de manutenção de uma só vez.

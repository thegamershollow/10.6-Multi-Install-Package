Verificar Preferências

Essa opção verifica a sintaxe dos arquivos de preferências (.plist).

Para verificar as preferências:

Abra o painel Verificar e clique na aba Preferências.

Selecione as preferências que deseja verificar (Usuário e/ou Sistema).

Caso queira visualizar somente os arquivos de preferência corrompidos, marque a opção "Mostrar apenas preferências corrompidas".

Clique o botão Verificar.

É recomendado apagar os arquivos de preferências corrompidos.

Clique num item para acessá-lo diretamente no Finder.

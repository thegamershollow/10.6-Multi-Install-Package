Cache do Sistema

É possível remover arquivos de cache das pastas do Sistema usando a aba Sistema, do painel Limpeza, ou no painel Automação. Todos os arquivos de cache usados pelo Sistema podem ser removidos sem risco. A eliminação desses arquivos recupera bastante espaço em disco e evita que arquivos corrompidos venham a comprometer o desempenho do computador.

O computador pode tornar-se mais lento após executar essa operação. Isso é normal, pois os caches tem que ser reconstruídos.

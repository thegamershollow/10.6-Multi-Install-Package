Logs

Quando um aplicativo está aberto, várias mensagens são trocadas entre ele, o sistema e seus componentes. Essa informação é armazenada em arquivos log. O sistema também cria logs e os arquiva durante as operações de manutenção. Quando um aplicativo não responde, detalhes são salvos em informes de diagnóstico que você pode usar para detectar o problema ou para enviá-lo a um desenvolvedor (caso lhe peçam).

a partir do painel Automação, é possível remover todos os logs e informes de diagnóstico, exceto os logs OnyX, bash e de Atualização do Sistema.

a partir da aba Logs, no painel Limpeza, é possível apagar diversos logs e informes de diagnóstico individualmente.

■

Arquivos de Log: essa opção apaga os logs dos aplicativos, de comandos do Terminal e alguns logs do Sistema.

■

Log de Atualização de Software da Apple: é possível apagar o arquivo de preferência que contém o log das Atualizações do Sistema usando essa opção. Esse arquivo (InstallHistory.plist) está localizado na pasta /Biblioteca/Receipts/ e pode ser visualizado nas Preferências do Sistema em "Atualização de Software" (aba Software Instalado).

■

Log de bash: é possível eliminar esse log, que contém uma lista com os comandos executados no Terminal, com essa opção.

■

Logs Arquivados do Sistema: ao executar os scripts periódicos (diário, semanal e mensal), logs são arquivados e comprimidos. Pode haver até sete arquivos por log. É possível remover todos os arquivos comprimidos e alguns logs salvos pelo Sistema usando essa opção.

■

Logs de mensagem instantânea: essa opção remove conversas salvas pelos aplicativos de mensagem instantânea mais populares. Suporte aos aplicativos: Adium, AIM, aMSN, Colloquy, Conversation, Fire, iChat, iRC, JBuddyMessenger, jIRCii, MacOSX.com Chat, MChat, Microsoft Messenger, Minerva, Proteus, Psi, Skype, Snak, SnapTalk, Trillian, VChat, X-Chat Azure e Yahoo! Messenger.

■

Informes de Uso do Usuário: use essa opção para apagar todos os Informes de Uso do Usuário. O script de manutenção também elimina esses arquivos, porém somente aqueles que não foram modificados nos últimos 60 dias.

■

Informes de Uso do Sistema: essa opção apaga todos os Informes de Uso do Sistema e reinicia o syslogd deamon (processo responsável por gravar atividades do sistema).

■

CrashReports de Aparelhos Móveis: essa opção elimina todos os Informes de Uso relatados pelos seus dispositivos móveis.

Visibilidade

Use o painel Visibilidade para alterar o estado de visibilidade de um volume, pasta ou arquivo.

Para modificar a visibilidade de um volume:

Selecione o volume pelo menu e então clique o botão Mostrar (para torná-lo visível) ou Ocultar (para torná-lo invisível).

Para modificar a visibilidade de uma pasta:

Clique o botão Mostrar Pasta para torná-la visível ou Ocultar Pasta para torná-la invisível, e então, ao aparecer a janela com as pastas, selecione aquela que deseja mostrar/ocultar.

Nota: Esses dois botões podem influenciar a visibilidade de um volume.

Para modificar a visibilidade de um arquivo:

Clique o botão Mostrar Arquivo para torná-lo visível ou Ocultar Arquivo para torná-lo invisível, e então, ao aparecer a janela com os arquivos, escolha aquele que deseja mostrar/ocultar.

Aviso: Todas essas operações reiniciam o Finder. Certifique-se de não haver operações em andamento (cópia/relocação de arquivos, gravação de CD/DVD ou esvaziamento do lixo).

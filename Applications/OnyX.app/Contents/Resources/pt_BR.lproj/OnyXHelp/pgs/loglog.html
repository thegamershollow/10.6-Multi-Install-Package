Log

O Log guarda data, hora e descrição de todas as operações executadas pelo OnyX. É possível escolher o número de linhas visualizadas ao digitar um número no campo de linhas e depois clicar o botão Recarregar. É possível também mostrar o arquivo numa janela do Finder e, a partir daí, apagar seu conteúdo.
Por padrão, o Log Onyx é apagado ao encerrar o aplicativo, mas é possível guardá-lo desmarcando essa opção nas preferências.

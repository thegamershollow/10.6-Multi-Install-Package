Temporary and Recent Items

You can use this option to remove the Temporary Items, the Recent Items in the Apple menu, the list of Recent Items in Disk Utility, the Recent Folders in the Finder's Go menu, the Recent Servers in Terminal, the Recent Places in the Opening and Saving dialogs, the Recent Items in the Open Recent command of the File menu of applications, and the appearance of Recent Items (files, folders, searches, servers, and so on) of third-party applications. Every time you connect to a computer, its address is stored in a file in the /Library/Recent Servers/ folder of your home folder. You can use this option to remove all these files.

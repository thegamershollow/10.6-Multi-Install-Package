You can delete various files using the Misc. tab of the Cleaning pane.

■

Recent items: You can use this option to remove the Recent Items in the Apple menu, the list of Recent Items in Disk Utility, the Recent Folders in the Finder's Go menu, the Recent Servers in Terminal, the Recent Places in the Opening and Saving dialogs, the Recent Items in the Open Recent command in the File menu of applications, and the appearance of Recent Items (files, folders, searches, servers, and so on) of third-party applications. Every time you connect to a computer, its address is stored in a file in the /Library/Recent Servers/ folder of your home folder. You can use this option to remove all these files.

■

Recent conversions of Calculator: You can use this option to delete the conversions recently used in Calculator.

■

Složku příloh aplikace Mail: This folder, situated in the Library folder of your home folder, contains a copy of files opened in the Mail application. To remove these copies, select this option.

■

Položky ve složce Uložená hledání: In the Finder's windows, you can save your searches as a Smart Folder. You can use this option to remove these saved searches located in the default folder.

■

Zastaralé položky: Under some conditions, a copy of the application state when an application quits unexpectedly is stored in core.xxx files located in the invisible /cores/ folder. You can use this option to remove all core.xxx files.

■

Předchozí knihovny iTunes: When you open a new version of iTunes, the existing library is saved in a "Previous iTunes Libraries" folder, then it is upgraded. If the library is upgraded correctly, the old library can be deleted with this option. This may make several MB of disk space available.

■

Seznam „Počítače” Sdílení obrazovky: You can use this option to delete the list located in the "My Computers" section of the Screen Sharing application.

■

QuickTime Content Guide: By default, launching QuickTime Player 7 displays the content listing in entertainment on the Internet (also available from the Show Content Guide command of the Window menu). You can use this option to delete the Cache files of this content.

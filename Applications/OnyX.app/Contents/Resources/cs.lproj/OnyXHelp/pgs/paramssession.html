Přihlášení

You can use the Login tab of the Parameters pane to set parameters for the miscellaneous options of the login window.

Background image:

You can change the background image of the login window.

■

Background: You can use the Select button to choose a replacement image for the background of the login window. By default, the application directs you to the folder containing the Desktop Pictures of Mac OS X (/Library/Desktop Pictures/...). You can choose another image, but it must be accessible by all users, for example in the Public, Shared, or Sites folder.
Formats accepted: PNG, JPEG, JPEG-2000, GIF, PSD, TIFF, TGA, and BMP.

■

Restore: You can use this option to put the original background image in the login window.

Message:

■

Zobrazovat zprávu v Okně přihlášení: You can use this option to display text in the login window. Select this option, then type the text in the field located below, and click the Apply button. Warning: This function doesn't accept certain special characters.

Různé volby:

■

Zakázat příkazy Restart, Vypnout a Odhlásit v nabídce Apple: You can turn the Kiosk mode on or off using this option. Turning on this mode disables the Restart, Shut Down, and Log Out commands in the Apple menu. Any modification of the Kiosk mode requires the restart of your computer.
Important: Use this option only if administrator accounts appear in the list of users.

■

Skrýt příkazy Spát, Restart a Vypnout v Okně přihlášení: This option hides the Sleep, Restart, and Shut Down buttons located at the bottom of the login window.

■

V Okně přihlášení zobrazovat: By default, the host name appears at the top of the login window. You can use this menu to display supplementary info (System version or build, serial number, IP address, Directory Services Status, or date and time).

■

V seznamu uživatelů zobrazovat: You can use this option to show or hide the users (Administrator, Local, and Network) in the list of users in the login window.
Warning: If the Kiosk mode is turned on, don't deselect the administrator accounts in the list of the users.

Režim spuštění počítače:

You can use this menu to set the startup mode of the computer.

■

Normální: Normal mode displays a dark gray Apple logo (the apple) on a light gray background, shows status messages above an Aqua progress indicator (a moving bar) while various initializations are carried out, and then displays the Desktop in the Finder.

■

Bezpečný režim: Safe Mode is the state Mac OS X is in after a Safe Boot. Starting up into Safe Mode forces a directory check of the startup volume, loads only required kernel extensions, runs only Apple-installed startup items, disables all fonts other than those installed by the system, moves all font caches to the Trash, and disables any Login Items. Safe Mode can be useful for troubleshooting. However, certain Mac OS X features do not work when you are started up in Safe Mode. For example, you can't use DVD Player, capture video in iMovie, use an AirPort card, use audio input or output devices, or use an internal or external USB modem.

■

Se záznamy: Verbose mode displays a black screen with white text that shows details of the startup process line by line, then displays the Desktop in the Finder.

■

Jednouživatelský: Single user mode displays a black screen with white text that shows details of the startup process line by line, then displays a command prompt; to return to Normal mode, type the command nvram boot-args = "" . If you don't like Terminal, forget the Single user mode.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by any other application. After execution, all parameters will be reset and display as they did when the System and the applications were first installed.

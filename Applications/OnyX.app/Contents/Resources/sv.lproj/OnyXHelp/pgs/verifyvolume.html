Verifiera startskivan

OnyX tillåter dig att kontrollera strukturen för systemfiler på startskivan och valet att köra denna kontroll vid start av OnyX. Innan du kör denna kontroll, kommer OnyX att avsluta alla öppna program (vissa program kan störa resultatet av denna kontroll.

Viktigt: Medans kontrollen pågår, kan Finder verka långsam eller inte svara. Snurrande "badbollen" kan visa sig eller så verkar det som att datorn inte svarar. Detta är normalt, var god vänta!

Varning: Denna kontroll är endast möjlig om hårddisken är journalförd.

Starta OnyX och mata in ditt administratörslösenord.

Klicka på ikonen Kontrollera och välj sedan fliken Volymstruktur.

Klicka på knappen "Kontrollera" för att omedelbart kontrollera startskivan.

Som standard frågar OnyX dig om du vill kontrollera volymstrukturen för startskivan vid start. Du kan välja att automatiskt kontrollera, eller inte göra någonting genom att välja lämpligt val i fliken Avancerat i OnyX inställningar.

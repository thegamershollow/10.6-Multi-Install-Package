Profil

Fliken Profil i panelen Info tillåter dig att visa eller spara en profil av din hård- och mjukvarukonfiguration.

3 typer av profiler finns:

- Miniprofil: Denna profil innehåller bara ett minimum av information och utelämnar datorns identifieringsinformation (serie nummer, IP adress...).

- Basprofil: Denna profil innehåller mesta hårdvaruinformationen och utelämnar mesta av mjukvaruinformationen.

- Full profil: Denna profil innehåller all hård- och mjukvaruinformation.

Varje profil kan sparas till Text formatet (.txt fil) eller XML formatet (.spx fil som kan läsas av programmet Systeminformation).

För att visa en profil:

Välj profiltypen du vill visa i lokala menyn "Profiltyp".

Klicka på knappen "Visa".

För att spara en profil:

Välj profiltypen du vill spara via lokala menyn "Profiltyp".

Välj önskat format (text eller xml) i lokala menyn "Format".

Klicka på knappen "Spara". Profilen sparas på ditt skrivbord.

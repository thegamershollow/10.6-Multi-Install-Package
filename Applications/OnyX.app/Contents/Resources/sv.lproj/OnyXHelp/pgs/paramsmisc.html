Diverse

Fliken Övrigt i panelen Parametrar tillåter dig att ändra gömda parametrar för vissa program.

Skivverktyg:

■

Verfiera skivavbilder (.dmg) när de monteras: Detta val tillåter dig att aktivera eller avaktivera verifiering av skivavbilder när de monteras.

■

Visa extra skivavbildsformat: Detta val tillåter dig att lägga till extra format om du vill konvertera eller skapa en skivavbild från en mapp. Denna funktion tillåter dig inte att skapa en ny tom skivavbild.

■

Visa Debug-meny: Välj detta om du vill visa en Debug meny i Skivverktyg's meny rad.

■

Verfiera skivor efter bränning: Detta val tillåter dig att aktivera eller avaktivera verifiering av skivor efter bränning.

Hjälpvisare:

■

Sätt fönster till flytande: Som standard visas Hjälpvisarens fönster som flytande, fönstret "flyter" så att säga överst, ovan alla andra fönster. Stäng av detta val för att visa Hjälpvisarens fönster som vanligt fönster, som kan hamna under andra fönster. När detta val är avstängt, så syns Hjälpvisarens ikon både i Dock och programväxlaren.

■

Visa ikon i Dock och programväxlaren: Som standard visas ingen ikon i Dock eller programväxlaren när Hjälpvisaren aktiveras. Detta val tillåter dig visa eller gömma Hjälpvisarens ikon i Dock och programväxlaren.

■

Aktivera debug läge: Detta läge lägger till mycket information i system loggen (/private/var/log/system.log) vid användning av Hjälpvisaren.

X11:

■

Visa varningsmeddelande innan X11 avslutas: Detta val tillåter dig att visa eller gömma varningsmeddelandet som visas som förval vid avslut av något X11 program.

■

Vid start av X11, kör: Denna meny tillåter dig att välja programmet som ska startas automatiskt när du startar X11.

Skärmdelning:

■

Visa Bonjour bläddrare: Detta val tillåter dig att visa eller gömma listan med tillgängliga maskiner via Bonjour protokollet.

■

Visa full verktygsrad: Detta val tillåter dig att visa eller gömma full verktygsrad vid skärmdelning.

Dashboard:

■

Aktivera Dashboard och dess widgetar: Om du aktiverar detta val, så visas dina widgets när du klickar på Dashboard ikonen i Dock.

■

Aktivera Dashboards utvecklarläge: Detta val tillåter dig att använda widgets från Dashboard på skrivbordet. Välj detta val och klicka på knappen "Genomför" (Dock startas om). Sedan i Apple menyn, välj Systeminställningar..., klicka på Exposé och Spaces ikonen och välj en tangentbordsgenväg för Dashboard.

- För att placera en widget på skrivbordet, klicka på Dashboard ikonen i Dock eller använd tangentbordsgenvägen du tidigare valde för att visa Dashboard. Klicka på den widget du vill använda och håll ned musknappen, använd sedan tangentbordsgenvägen för att gömma Dashboard. Widgeten kommer att stanna kvar på Skrivbordet och kan användas som vilket program som helst. Du kan placera så många widgets du vill på ditt skrivbord.

- För att ta bort en widget från skrivbordet, klicka på widgeten och håll ned musknappen använd sedan tangentbordsgenvägen för att visa Dashboard. Släpp sedan musknappen och använd tangentbordsgenvägen igen för att gömma Dashboard.

■

Stäng av vågeffekten vid tillägg av ny widget: Detta val tillåter dig att aktivera eller avaktiver vågeffekten som syns när du lägger till en ny widget i Dashboard.

Andra program:

■

Apple Remote Desktop: Detta val visar eller gömmer en debug meny i Apple Remote Desktop's meny rad.

■

iCal: Detta val visar en debug meny i iCal's meny rad.

■

iChat: Detta val aktiverar eller avaktiverar data detektorer i iChat. Data detektorer är en mekanism som tillåter iChat att känna igen vissa typer av information och föreslå passande operation från en lokal meny.

■

Terminal: Aktivera Terminalens fönsterfokusering, vilket aktiverar ett terminal fönster genom att placera muspekaren ovan för fönstret.

■

Interface Builder: Detta val, som är avaktiverat som standard från version 3.2, tillåter dig att aktivera AppleScript Inspector i Interface programmet.

Klicka på knappen "Återställ förval" för att avbryta alla modifieringar av OnyX eller annat program. Efteråt, så kommer parametrarna att vara nollställda och visningen att vara samma som då Systemet installerades för första gången.

Problem med "Öppna med" kontextmenyn i Finder

Om "Öppna med" kontextmenyn i Finder visar dubletter av program eller program som inte är installerade längre, kan OnyX hjälpa dig att korrigera problemet genom att bygga om LaunchServices databasen.

Kör OnyX.

Öppna Automatisera- eller Underhållspanelen och välj fliken Bygg om...

Markera valet "LaunchServices" och avmarkera alla andra val.

Klicka på knappen Genomför.

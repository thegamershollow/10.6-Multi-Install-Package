locate databas / Hitta...

locate är ett Unix kommando som tillåter dig att snabbt hitta en fil eller mapp genom att söka i locate databasen, som är ett index över alla lagrade objekt på hårddisken. Vid installation av systemet, så finns inte databasen och du måste skapa den och uppdatera den regelbundet om du vill använda locate kommandot.


Panelen Verktyg > fliken Hitta:

I panelen Verktyg, fliken Hitta, tillåts du skapa, ta bort och uppdatera locate databasen. När väl databasen är skapad, kan du söka efter objekt. Klicka på ett objekt i listan för att visa den i Finder. Denna flik visar även storleken på databasen och datum för senaste uppdatering.

Panelen Underhåll > fliken Bygg om:

Om locate databasen redan har skapats, kan den uppdateras från fliken Bygg om i panelen Underhåll.

Varning: Skapa och bygga om locate databasen kan ta lång tid. Var god vänta!

Allmänt

Fliken Allmänt tillåter justering av en del globala parametrar.

Fönster:

■

Pilarna i rullisten: OnyX tillåter dig att välja mellan fyra olika typer av pilar i fönstrens rulllister. Välj något av dessa val för att ändra inställningarna direkt och starta om Finder. Program som redan körs, måste startas om för att visa ändringarna.

■

Hastighet för visning av blad (sheets): Detta val tillåter dig att välja en hastighet för visning av ett fönsters "blad". Ett "blad" är en dialogruta som tillhör ett fönster och dyker fram när användaren frågas om inmatning av något slag.

Krashrapporteraren:

■

Krashrapporterns funktionssätt: När ett program krashar, så visas ett varningsfönster där du kan starta om programmet och skicka en rapport till Apple. Det är standard funktionen men det finns två andra lägen. Utvecklarläget visar ett varningsmeddelande med diverse debug information. Serverläget (standard i Mac OS X Server) visar inget meddelande, utan lagrar krashrapporterna i /Library/Logs/CrashReporter och i Library/Logs/CrashReporter i din hemmapp.

Nätverksvolymer:

■

Förhindra att .DS_Store filer skapas: Detta val tillåter dig att förhindra att .DS_Store filer skapas på nätverksvolymer. Som förval är dessa filer osynliga i Mac OS X men de syns (och är oanvändbara) på andra operativsystem.

Skärmavbilder:

■

Skärmavbildsformat: Denna meny tillåter dig att ändra formatet (PNG som förval) på skärmavbilder tagna med tangetbordsgenvägarna. Du kan välja mellan nio format: BMP, GIF, JPEG, JPEG 2000, OpenEXR, PDF, PNG, TGA och TIFF.

■

Skugga i skärmavbildningar: När du fångar ett fönster (Shift + Command + 4 sedan mellanslagstangenten), så fångas som förval även fönstrets skugga. Detta val tillåter dig att förhindra att skuggan tas med i skärmavbilden.

■

Sökväg: Detta val specifierar sökvägen där skärmavbilderna lagras (användarens skrivbord som förval). För att ändra, klicka på knappen "Välj...", välj sedan en mapp och klicka på "Välj".

■

Namn: Du kan välja namnet för skärmavbilder genom att mata in önskat namn i Namn fältet och klicka på knappen "Utför".

Senaste objekt:

■

Visa programs senaste objekt i menyn Arkiv > Öppna senaste: Majoriteten av program som kan öppna dokument har en "Öppna senaste" meny i Arkiv menyn. Som standard visar denna undermeny de tio senaste öppnade dokumenten. Denna funktion tillåter dig att aktivera eller avaktivera denna funktion. Dock finns det program (Photoshop, Golive, GraphicConverter...) som använder sina egna funktioner för att lagra senaste objekt.

■

Antal senaste objekt: Detta fält tillåter dig att välja antal "platser" i Arkiv > Öppna senaste menyn för program som tillåter det.

■

Antal senaste platser i Öppna/Spara dialoger: Denna lokala meny tillåter dig att välja 1 - 12 senaste platser (standard är 5) i Öppna/Spara dialoger.

Animerad skrivbordsbakgrund:

■

Animerad skrivbordsbakgrund: Detta val tillåter dig att välja en skärmsläckare av de installerade i Mac OS X och starta den som en skrivbordsbakgrund.

Nedladdade program:

■

Stäng av "första start" varningsmeddelande för nedladdade program: Vid första start av ett nedladdat program, så visas ett varningsmeddelande. Detta meddelande visar diverse information om var den laddades ned och eventuellt avbryta start av programmet. Detta val tillåter dig att visa eller gömma detta varningsmeddelande.

Inställningspaneler:

■

Arkiv: Denna knapp installerar inställningspanelen "Arkiv" som tillåter ändring av inställningar för programmet "Arkivverktyg".

■

Processor: Denna knapp installerar inställningspanelen "Processor".

För att ta bort en inställningspanel, högerklicka på dess ikon och välj "Ta bort ...".

Klicka på knappen "Återställ förval" för att avbryta alla modifieringar av OnyX eller annat program. Efteråt, så kommer parametrarna att vara "nollställda" och visningen att vara samma som då Systemet installerades för första gången.

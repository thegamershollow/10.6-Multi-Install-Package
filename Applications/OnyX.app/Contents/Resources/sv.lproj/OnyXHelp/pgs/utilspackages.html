Paket

Alla Apple program och vissa tredjeparts program installeras från ett paket (.pkg). Vid installation skapas en receptfil (.bom) och lagras i en gömd systemmapp (/private/var/db/receipts).

Fliken Paket i panelen Verktyg tillåter dig att öppna dessa filer (.bom) och visa alla objekt som installerades av ett paket. Klicka på knappen "Visa i Finder" för att se receptfilen (.bom) i Finder.

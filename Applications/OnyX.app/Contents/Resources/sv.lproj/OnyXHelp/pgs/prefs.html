Inställningar

För att visa OnyX inställningar, välj "Inställningar..." i OnyX menyn. Inställningsfönstret visar fem flikar.

För att ändra utseendet för programmet:

Fliken Gränssnitt tillåter dig att ändra OnyX's utseende.

■

Välj Metall eller Aqua för utseendet för huvudfönstret. Denna ändring får effekt vid nästa start av OnyX.

■

Aktivera "låd-effekt" vid öppning av OnyX paneler.

■

En lokal meny tillåter dig att välja en panel som ska öppnas automatiskt vid start av OnyX.

■

Du kan reducera ogenomskinligheten av huvudfönstret genom att dra i reglaget (får effekt direkt).

■

En lokal meny tillåter dig att välja en annan ikon för OnyX och en andra meny tillåter dig att ändra ikon för verktygsraden.

För att ändra programmets funktionalitet:

Fliken Beteende tillåter dig att ändra hur OnyX fungerar.

■

Du kan välja att köra kommandon för nuvarande konto (förval) eller för alla konton om möjligt.

■

När en omstart är rekommenderad, visar OnyX som standard ett meddelande som frågar dig vad du vill göra. Du kan välja att inte göra någonting, starta om eller stänga av din Mac.
Alternativet Tvinga omstart/avstängning, startar om eller stänger av din dator utan att visa någon varning. Om ett dokument inte är sparat, kommer du att förlora de senaste ändringarna.

För att uppdatera programmet:

Fliken Uppdatering tillåter dig att ställa in OnyX att automatiskt kontrollera efter uppdateringar och visar datum och tid för senaste uppdateringskontrollen.

■

För att kontrollera vid start om det finns en uppdatering, välj "Leta efter uppdateringar vid start". Om en nyare version är tillgänglig, så sparas den till skrivbordet (förval).

■

Du kan ändra sökväg där uppdateringar ska sparas: klicka på knappen "Välj..." och välj en annan mapp.

■

Klicka på knappen "Kontrollera" för att kolla om det finns någon uppdatering.

Visa meddelanden och OnyX logg:

Fliken Meddelande och Logg tillåter dig att se olika meddelanden och loggen för OnyX.

■

Alternativet "Visa viktig information" visar viktig information vid start som användaren bör veta om innan användning av OnyX.

■

Vid start, visar OnyX som standard programlicensavtalet. Efter ha läst och accepterat det, så kan du stänga av visning av detta genom att avmarkera "Visa licensinformation".

■

Välj "Visa meddelande > Innan program avslutas" om du vill bli meddelad innan OnyX tvångsavslutar ett program.

■

Alternativet "Efter kommandon genomförts" visar ett meddelande efter varje körd operation.

■

Vid avslut av OnyX, visar programmet som standard ett bekräftelsemeddelande, som du kan välja att stänga av.

■

Som standard raderas OnyX loggen vid avslut men avmarkering av detta val så behålls loggarna.

Avancerat:

Fliken Avancerat tillåter dig att ställa in kontrollalternativen när OnyX startas. Det finns tre tillgängliga alternativ (fråga, kontrollera automatiskt eller kontrollera inte).

■

Kontrollera hårddiskens S.M.A.R.T-status: Detta val genomför diagnostiska tester som kan förhindra hårddiskhaveri. Det fungerar endast med ATA eller Seriell ATA hårddiskar. FireWire, USB och SCSI hårddiskar stöds inte i Mac OS X. OnyX kan inte kontrollera S.M.A.R.T. status för RAID arrayer.

■

Kontrollera startskivan vid start: Eftersom vissa program kan störa denna operation, kommer OnyX att avsluta alla öppna program. Under kontrollen kan Finder bli långsam eller sluta svara. Snurrande "badbollen" kan synas eller så kan datorn tyckas slutat fungera. Detta är normal, så var god vänta!
Var god notera att detta kontrollalternativ bara är möjligt när hårddisken är journalförd.

Klicka på "Återställ förval" för att avbryta dina ändringar och återställa OnyX till förvald inställning.

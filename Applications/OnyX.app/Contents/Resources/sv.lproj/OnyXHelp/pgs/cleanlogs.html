Loggar

När ett program körs så skapas många meddelanden mellan programmet, systemet och dess komponenter. Denna information lagras i loggar. Systemet skapar också loggar och arkiverar dem vid systemunderhåll. När ett program krashar sparas information om detta i en diagnostisk rapport som du kan använda för att diagnostisera problemet eller skicka till utvecklaren (om tillfrågad).

Panelen Automatisera tillåter dig att radera alla loggar och diagnostiska rapporter utom för OnyX, bash och loggarna för systemets programuppdatering.

Loggfliken i panelen Rensa tillåter dig att radera dem individuellt.

■

Loggar: Detta val raderar loggarna för program, terminalen och vissa systemloggar.

■

Programuppdaterings logg: Detta val raderar loggfilen för Apple programuppdatering. Denna logg (InstallHistory.plist) finns i /Library/Receipts mappen och kan ses i "programuppdatering" inställningspanelen i fliken "installerade uppdateringar".

■

bash logg: Detta val raderar loggen för inmatade kommandon i Terminal programmet.

■

Arkiverade loggar: När underhållsskripten (dagligen, veckovis och månadsvis) körs, arkiveras och komprimeras loggar. Det kan vara upp till sju arkiv per logg. Detta val raderar alla komprimerade arkiv och vissa loggar sparade av systemet.

■

Konversations-loggar från chat-program: Detta val raderar sparade konversationer för de populäraste chat programmen. Program som stöds: Adium, AIM, aMSN, Colloquy, Conversation, Fire, iChat, iRC, JBuddyMessenger, jIRCii, MacOSX.com Chat, MChat, Microsoft Messenger, Minerva, Proteus, Psi, Skype, Snak, SnapTalk, Trillian, VChat, X-Chat Azure och Yahoo! Messenger.

■

Användardiagnostiska rapporter: Detta val raderar alla användardiagnostiska rapporter. Det dagliga underhållsskriptet radera också dessa filer men endast de som inte har ändrats de senaste 60 dagarna.

■

Systemdiagnostiska rapporter: Detta val raderar diagnostiska rapporter skapade av systemet och startar om syslogd daemon (process som övervakar systemets aktiviteter).

■

Krashrapporter för mobila enheter: Detta val raderar de diagnostiska rapporter som har uppstått på din mobila enhet.

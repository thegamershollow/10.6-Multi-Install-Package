LaunchServices

Detta val bygger om LaunchServices databasen, som innehåller vilka program som kan öppna vilka dokument som förval.

När du startar ett nytt slags dokument för första gången, så visas en varningsdialog. Du har då flera val: tillåt dokumentet att öppnas av programmet som systemet har som förval, låta Finder visa detta program i dess mapp eller avbryta öppning av dokumentet. När dokumentet väl har öppnats en gång, så lagras kopplingen mellan dokumenttypen och programmet. Det som också lagras är alla installerade program, dina specifika val för öppna dokument med specifika program samt ditt val av förvald webbläsare.

Efter ombyggnad, så kan dubbelklickning av dokument eller webblänkar starta oväntade program. För att ändra detta till dina föredragna inställningar, välj en fil, välj "Arkiv/Visa info" (i Finder) eller Apple-I (tangentbordet) och välj bland programmen i listan "Öppna med", klicka sedan "Ändra alla" för att säga till Finder att öppna alla liknande filer med samma program. Förvald webbläsare kan ställas in i webbläsarens inställningar. Ombyggnad av LaunchServices kan även korrigera vissa problem: dubletter eller raderade program i Finders "Öppna med" kontextmeny, felaktiga ikoner för vissa filer och mappar, start av fel program vid dubbelklick av dokument.

LaunchServices kan byggas om från panelen Underhåll > Bygg om eller från panelen Automatisera.

Varning: Ombyggnad av LaunchServices databasen kan ta lång tid att genomföra. Var god vänta!

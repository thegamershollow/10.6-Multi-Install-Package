Fel vid kontroll av startskivan

Om OnyX upptäcker ett fel vid kontroll av diskens struktur, reparera detta så snart som möjligt.

För att reparera en disk:

Avsluta OnyX.

Mata in Mac OS X installationsskivan, starta om datorn, håll ned "C" tangenten efter uppstartsljudet för att starta från CDn.

Välj ditt språk.

I Verktygsmenyn, välj Skivverktyg.

I vänstra kolumnen, välj din startskiva.

I S.O.S. fliken till höger, klicka Reparera skivan och vänta.

Starta sedan om normalt...

För det mesta så är felen vanligtvis ingenting allvarligt och korrigeras vanligvis av Skivverktyg. Dock kan mera allvarliga fel kräva användning av tredje parts program (Drive Genius, DiskWarrior, TechTool Pro...).

Kontrollera hårddiskens S.M.A.R.T. status

OnyX tillåter dig att kontrollera hårddiskens S.M.A.R.T. status men även att köra denna kontroll vid start av OnyX. Detta val genomför en diagnostik som kan förhindra en hårddiskhaverering. De flesta hårddiskarna inkluderar S.M.A.R.T. hårdvara (Self-Monitoring Analysis and Reporting Technology) som testar och övervakar hårddisken.

Varning: Kontroll av S.M.A.R.T. status för startskivan fungerar endast med en ATA eller Seriell ATA hårddisk. FireWire, USB och SCSI diskar stöder (ännu) inte detta i Mac OS X. OnyX kan inte kontrollera S.M.A.R.T. status för ett RAID set.
- If FileVault is turned on, checking the S.M.A.R.T. status will not be possible (Message: The S.M.A.R.T. status is not supported.). If you turn off FileVault, restart the computer again before checking the S.M.A.R.T. status of the disk.

Starta OnyX och mata in ditt administratörslösenord.

Klicka på ikonen Kontrollera, välj sedan fliken S.M.A.R.T.-status.

Klicka på knappen "Kontrollera" för att omedelbart kontrollera startsskivans S.M.A.R.T. status.

Fliken S.M.A.R.T.-status i panelen "Kontrollera", visar S.M.A.R.T. status och datum för senaste kontroll.

Som standard frågar OnyX dig om du vill kontrollera S.M.A.R.T. status för startskivan vid start. Du kan välja att automatiskt kontrollera eller inte genom att välja inställning i fliken Avancerat i OnyX inställningar.

Program

Fliken Program tillåter dig att köra program som finns i Mac OS X men inte lätt att komma åt för användaren.

Skärmdelning:

Med Mac OS X kan du ansluta till en annan dator på ditt nätverk och visa den datorns skärm på din dator. Medan du delar datorns skärm, kan du kontrollera vad som händer på den datorn, som att öppna dokument eller program, öppna, flytta eller stänga fönster och även stänga av datorn.

Du kan också dela din dators skärm med en annan dator på ditt nätverk.

Nätverksdiagnostik:

Du kan använda Nätverksdiagnostik för att hjälpa att lösa anslutningsproblem till internet. Om du inte kan ansluta till internet, öppnas Nätverksdiagnostik och hjälper dig att lösa problemet.

Du kan använda Nätverksdiagnostik för att diagnostisera problem med ditt Ethernet, AirPort eller internt/externt modem. Om du ansluter till internet på något annan sätt, se dokumentationen för ditt internet konto.

Teckenvisaren:

Använd Teckenvisaren för att mata in speciella tecken och symboler, så som matematiska symboler, pilar eller andra knasigheter i dina dokument.

Du kan också använda Teckenvisaren för att mata in japanska, kinesiska och koreanska tecken samt tecken från andra språk.

Teckenvisaren är tillgänglig i de flesta program från Redigera > Specialtecken menyn.

Katalogverktyg:

Katalogverktyg avgör vilken katalogservice en Mac OS X dator använder och hur den ansluter till specifika katalogdomäner. Katalogverktyg definierar också sökalternativen för hittande av autentisering och kontaktinformation i specifika katalogdomäner.

Se Katalogverktyg's hjälp för mer information.

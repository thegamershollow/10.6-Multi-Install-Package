Scripts de maintenance

Mac OS X intègre un mécanisme de gestion de tâches (launchd), qui permet d'automatiser des processus de nettoyage du Système en exécutant un utilitaire spécifique (periodic). Cet utilitaire lance plusieurs scripts BSD standards qui effectuent de nombreuses opérations de maintenance.

Pour exécuter les scripts de maintenance :

Ouvrez le panneau "Maintenance" et cliquez sur l'onglet "Scripts de maintenance".

Sélectionnez les scripts que vous souhaitez exécuter (quotidiens, hebdomadaires et/ou mensuels).

Cliquez sur le bouton "Exécuter".

Note : Le panneau Automation vous permet également d'exécuter tous les scripts de maintenance.

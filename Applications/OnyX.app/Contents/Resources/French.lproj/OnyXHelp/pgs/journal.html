Journalisation

Cette option permet d'activer ou de désactiver la journalisation du disque.

Si vous activez la journalisation d'un disque, Mac OS X enregistre continuellement toutes les modifications intervenues sur les fichiers du disque. Si votre ordinateur s'arrête à cause d'une coupure d'électricité ou pour toute autre raison, Mac OS X se sert du journal pour restaurer le disque dur grâce au dernier état stable disponible avant l'arrêt.

Note : Si la journalisation est désactivée, il n'est pas possible de vérifier la structure du système de fichiers du volume de démarrage.

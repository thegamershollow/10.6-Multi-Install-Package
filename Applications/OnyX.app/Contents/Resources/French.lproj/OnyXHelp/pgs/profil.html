Profil

L'onglet Profil du panneau Informations vous permet d'afficher ou d'enregistrer le profil de votre configuration matérielle et logicielle.

Il existe 3 types de profil :

- Mini Profil : Ce profil ne contient qu'un minimum d'informations omettant les informations d'identification de votre ordinateur (numéro de série, adresse IP...).

- Profil de base : Ce profil contient la plupart des informations matérielles omettant la majorité des informations logicielles.

- Profil complet : Ce profil contient toutes les informations sur votre configuration matérielle et logicielle.

Chaque profil peut être enregistré au format Texte (fichier .txt lisible par TextEdit ou tout autre éditeur de texte) ou au format XML (fichier .spx lisible par l'application Informations Système d'Apple).

Pour afficher un profil :

Sélectionnez le profil que vous souhaitez visualiser dans le menu local "Type de profil".

Cliquez sur le bouton "Afficher".

Pour enregistrer un profil :

Sélectionnez le profil que vous souhaitez enregistrer dans le menu local "Type de profil".

Sélectionnez le format d'enregistrement (Texte ou XML) dans le menu local "Format".

Cliquez sur le bouton "Enregistrer". Le fichier Profil est enregistré sur le Bureau.

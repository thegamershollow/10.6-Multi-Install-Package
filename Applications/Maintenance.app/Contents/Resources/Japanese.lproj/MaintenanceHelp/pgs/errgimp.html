GIMP doesn't run anymore

If GIMP doesn't run anymore and seems blocked on a message displaying "Caching Fonts", you have probably removed the Fonts Cache.

■

In this case, don't force GIMP or X11 to quit; wait for the Fonts Cache to rebuild.

Indeed, the rebuilding of Fonts Cache in some versions of GIMP is particularly long.
Once the Cache has been rebuilt, GIMP will open again normally.

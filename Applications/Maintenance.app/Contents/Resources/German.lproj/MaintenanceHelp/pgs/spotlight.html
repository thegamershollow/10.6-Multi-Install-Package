Spotlight Index

Spotlight, the intelligent search function of Mac OS X, is very fast because it creates and uses an index of all files on all connected hard disks. This option removes this index. A few seconds after deleting it, the system will automatically rerun the content indexing of the hard disk in the background. You can see this progress by clicking the Spotlight icon in the menu bar. Nevertheless, you can restart your computer, the indexing will be stopped, and will continue after a restart.

If indexing is turned off, removing the index will reactivate it automatically.

Script di manutenzione

Mac OS X includes a task manager (launchd) which automates System cleaning by executing the "periodic" utility. This utility runs three BSD standard scripts which effect numerous maintenance operations.

Theoretically, since version 10.4 of Mac OS X, these scripts don't run anymore at precise hours and days but only when the number of effective hours or days is passed. In practice, restart, shut down, and sleep seem to disturb the functioning of launchd, and it's recommended to manually run them from time to time.

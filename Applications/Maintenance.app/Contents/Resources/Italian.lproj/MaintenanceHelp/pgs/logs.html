Resoconti e Resoconti di blocco

When an application is open, it sends messages between the application, system, and its components. This information is stored in log files. The system also creates logs and archives them during System maintenance operations. When an application does not respond, the details are saved in a Log file that you can use to diagnose the problems, or to send it (if necessary) to the developer. This option deletes any Diagnostic Reports created by the system and relaunches the syslogd daemon (process that records the activities of the system).

Maintenance allows you to remove these logs, the compressed archives, the CrashReporter, and the following web browsers' History: Safari, Firefox (dapprima Firebird e Phoenix), Camino (dapprima Chimera), Shiira, OmniWeb, Cruz, Navigator (Netscape), Opera, iCab, Flock, Fake, SeaMonkey (dapprima Mozilla), Dreams, Demeter, KidsBrowser, NagaraBrowser, MiniBrowser, WebBackdrop, Hana, BumperCar, Scavenger, TrailBlazer, Sunrise, Xyle scope, surfDude, amaya, iBrowser, Scourge, Gmail Browser, Lightbrowser, WebDesktop, Internet Explorer, Desktop Web Browser, Navigateur Web, WebKit, DeskBrowse, Madfox, Webstractor, Nova X (dapprima RAFBrowseX), Radon (dapprima Tominated Browser), MiaBrowser, wKiosk, Wyzo, KidZui, Stainless, WamCom, Chromium, Google Chrome, Roccat, TenFourFox, iTubeX, and SRWare Iron.

Note: The Software Update Log is not deleted.

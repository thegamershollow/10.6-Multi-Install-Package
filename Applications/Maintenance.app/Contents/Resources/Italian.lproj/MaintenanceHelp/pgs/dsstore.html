File nascosti .DS_Store

An invisible file (.DS_Store) maintains various information regarding the folder in which it is located (icon view and positions). On other systems, those files are visible. You can choose to remove them before burning a CD-ROM, for example... This option deletes all .DS_Store files from the hard disk. After deleting these files, the appearance of content of your folders will be the same as when the System was first installed.

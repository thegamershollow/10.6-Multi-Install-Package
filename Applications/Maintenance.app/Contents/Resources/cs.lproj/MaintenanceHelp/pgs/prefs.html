Preferences

To display the Preferences window, choose Preferences in the Maintenance menu. This window displays four tabs.

To modify the appearance of the application:

The Interface tab allows you to modify the appearance of the main window. These modifications are immediately visible.

■

Choose Metal or Aqua to change the background.

■

You can also add transparency to the main window by moving the slider.

To modify application behavior:

The Behavior tab acts on the global functioning of the application.

■

You can choose to execute the maintenance on the current account or all accounts.

■

By default, when a restart is recommended after running certain commands, Maintenance displays a message which asks what you want to do. You can choose to do nothing, restart, or shut down the Mac.
The Force restart/shut down option restarts or shuts down the computer without displaying an alert message. If a document is not saved, you will lose the last modifications.

To update the application:

The Update tab allows you to search, when Maintenance opens, whether a more recent version of Maintenance is available. This tab displays the date and the last search.

■

To search for an update when Maintenance opens, select "Check for update when Maintenance opens." Then, you can download it to be placed on your Desktop.

■

Click the Check button to immediately check whether a more recent version of Maintenance is available.

Advanced Preferences:

The Advanced tab allows you to verify and to set checks when Maintenance opens.

■

Verify the structure of System File of the startup volume. You can choose to automatically check it or not by selectingMaintenance will quit all the corresponding option. Before running this verification, Maintenance quit all the open applications. (Some tasks can disturb the result of checking.) During the verification, the Finder may act slow or become unresponsive, the wait cursor (which looks like a colored pinwheel) may appear, or your computer could appear to stop responding (freeze). It's normal, so wait for the verification to finish!
Warning: This verification is only possible if the disk is not journaled.

■

Checking the S.M.A.R.T. status of the disk performs a diagnostic test to detect drive failure. Most disks include S.M.A.R.T. hardware (Self-Monitoring Analysis and Reporting Technology) that tests and monitors the drive. Select this option if you want to check the disk when Maintenance opens. Click the Check button to immediately verify the S.M.A.R.T. status of the startup disk.
Warning: Checking the S.M.A.R.T. status of the startup disk is only possible with an ATA or Serial ATA disk. The FireWire, USB, and SCSI disks cannot be checked using Mac OS X 10.6.x.

Click on Restore Defaults to cancel your modifications.

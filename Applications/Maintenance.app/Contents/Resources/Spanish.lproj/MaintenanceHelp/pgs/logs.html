Registros e Informes de Fallos

Cuando una aplicación está ejecutándose, esta envia mensajes entre la aplicación, el sistema y sus componentes. Estas informaciones son almacenadas en los ficheros de Registro. El Sistema también crea registros y archivos propios durante las operaciones de mantenimiento. Cuando una aplicación se bloquea, los detalles son guardados en un fichero de registro que puede usar para diagnosticar los problemas, o para enviarlo (si es necesario) al desarrollador. This option deletes any Diagnostic Reports created by the system and relaunches the syslogd daemon (process that records the activities of the system).

Maintenance le permite borrar todos los registros, informes de fallos e historial de Navegadores según: Safari, Firefox (anteriormente Firebird y Phoenix), Camino (anteriormente Chimera), Shiira, OmniWeb, Cruz, Navigator (Netscape), Opera, iCab, Flock, Fake, SeaMonkey (anteriormente Mozilla), Dreams, Demeter, KidsBrowser, NagaraBrowser, MiniBrowser, WebBackdrop, Hana, BumperCar, Scavenger, TrailBlazer, Sunrise, Xyle scope, surfDude, Amaya, iBrowser, Scourge, Gmail Browser, Lightbrowser, WebDesktop, Internet Explorer, Desktop Web Browser, Navigateur Web, WebKit, DeskBrowse, Madfox, Webstractor, Nova X (anteriormente RAFBrowseX), Radon (anteriormente Tominated Browser), MiaBrowser, wKiosk, Wyzo, KidZui, Stainless, WamCom, Chromium, Google Chrome, Roccat, TenFourFox, iTubeX, and SRWare Iron.

Note: The Software Update Log is not deleted.

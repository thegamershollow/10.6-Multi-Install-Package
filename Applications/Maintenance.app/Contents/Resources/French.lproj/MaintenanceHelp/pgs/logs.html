Historiques et rapports de problèmes

Lorsque une application est en cours d'exécution, elle envoie des messages entre les applications, les systèmes et les composants. Ces informations sont stockées dans des historiques. Le système créé également plusieurs historiques et les archive durant les opérations de maintenance du système. Lorsqu'une application se bloque, le détail du blocage est enregistré dans un fichier (rapport de problème) que vous pouvez utiliser pour diagnostiquer les problèmes liés à cette application, ou pour l'envoyer (si nécessaire) au développeur de l'application. Cette option supprime les éventuels rapports de diagnostic créés par le Système et relance le démon syslogd (processus qui enregistre les activités du système).

Maintenance permet de supprimer tous les historiques, les archives compressées, les rapports de problème ainsi que les historiques des navigateurs Web suivants : Safari, Firefox (anciennement Firebird puis Phoenix), Camino (anciennement Chimera), Shiira, OmniWeb, Cruz, Navigator (Netscape), Opera, iCab, Flock, Fake, SeaMonkey (anciennement Mozilla), Dreams, Demeter, KidsBrowser, NagaraBrowser, MiniBrowser, WebBackdrop, Hana, BumperCar, Scavenger, TrailBlazer, Sunrise, Xyle scope, surfDude, amaya, iBrowser, Scourge, Gmail Browser, Lightbrowser, WebDesktop, Internet Explorer, Desktop Web Browser, Navigateur Web, WebKit, DeskBrowse, Madfox, Webstractor, Nova X (anciennement RAFBrowseX), Radon (anciennement Tominated Browser), MiaBrowser, wKiosk, Wyzo, KidZui, Stainless, WamCom, Chromium, Google Chrome, Roccat, TenFourFox, iTubeX, and SRWare Iron.

Note : L'historique des mises à jour de logiciels Apple n'est pas supprimé.
